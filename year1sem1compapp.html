Computer Applications

COM-111 Resources & Study Guide
Computer
An electronic device through which data can be stored, processed, and then retrieved again when requested as information. It consists of Hardware and Software.
Data
The facts fed to the computer in order to process it to benefit from it by the user. data can be numbers, letters, symbols, images, or sounds.
Information
Data that has been processed and analyzed to provide knowledge to decision makers and help them achieve certain goals.
Processing
Performing some manipulations on the entered data to give the desired results by the program. This includes storage, arrangement, classification, search, and solving mathematical problems.
Knowledge
Awareness and understanding of a set of information.
First Generation Computers (1940-1956)
Computers that used vacuum tubes. They were bulky, consumed lots of electricity, generated much heat, and used machine language.
Second Generation Computers (1956-1963)
Computers that used transistors. They were smaller, faster, cheaper, and used assembly/high-level languages like FORTRAN and COBOL.
Third Generation Computers (1964-1971)
Computers that used Integrated Circuits (ICs), or silicon chips. Introduction of minicomputers. drastically increased speed and reliability.
Fourth Generation Computers (1971-Present)
Computers using Microprocessors (LSI and VLSI). Components of the computer were located on a single chip. Introduction of personal computers (PCs) and networks.
Fifth Generation Computers
Computers based on Artificial Intelligence (AI), capable of voice recognition, learning behavior, and decision making.
Vacuum Tubes
Fragile glass devices used in First Generation computers. They were unreliable and large.
Transistors
Semiconductor devices that replaced vacuum tubes. They amplify signals or open/close circuits. Invented at Bell Labs.
Integrated Circuits (IC)
A silicon chip containing many transistors, registers, and capacitors. Replaced discrete transistors in Third Generation.
Microprocessor
A single chip containing all the circuits required to perform arithmetic, logic, and control functions. Used in Fourth Generation.
Hardware
The physical components of a computer system that you can touch (e.g., CPU, Monitor, Keyboard, Hard Disk).
Software
Non-physical programs and instructions that drive the computer to perform tasks.
CPU (Central Processing Unit)
The "brain" of the computer. A chip consisting of electronic circuits that executes instructions, controls inputs/outputs, and processes data.
ALU (Arithmetic and Logic Unit)
Part of the CPU that performs all arithmetic (add, sub, mul, div) and logical operations (AND, OR, NOT, comparisons).
CU (Control Unit)
Part of the CPU that directs the flow of data and instructions within the computer. It controls all parts and delivers instructions.
Registers
High-speed storage locations inside the CPU for temporary data manipulation.
Data Bus
A set of wires that transfer data across the various units of the computer.
Addressing Bus
A set of wires that transfer the addresses of main memory cells specified by the processor.
Control Bus
A set of wires that transfer control commands from the CPU to other components.
Main Memory (Primary Storage)
Memory that the CPU accesses directly. Includes RAM and ROM.
RAM (Random Access Memory)
Volatile temporary memory used by the computer to store data and programs currently in use. Data is lost when power is off. It has larger capacity than ROM but is temporary.
ROM (Read Only Memory)
Non-volatile permanent memory that stores startup instructions (BIOS). Data is set by the manufacturer and cannot be erased or written to by normal users.
Secondary Storage
Non-volatile storage used to store data and programs permanently. Examples: Hard Disk, Floppy Disk, CD/DVD, Flash Memory.
Hard Disk (HDD)
A magnetic storage device with huge capacity and high access speed, usually installed inside the computer case.
Floppy Disk
An older magnetic storage medium with limited capacity (1.44 MB) and low speed.
Flash Memory
A portable solid-state storage device with large capacity and high speed, connects via USB.
CD-ROM
Optical disc that stores data using laser technology.
Input Units
Devices used to enter data and instructions into the computer (e.g., Keyboard, Mouse, Scanner, Microphone).
Output Units
Devices that display or produce the results of processing (e.g., Monitor, Printer, Speaker).
System Software
Programs that manage and control the computer hardware and provide a platform for application software. Includes Operating Systems and Compilers.
Application Software
Programs designed to help users perform specific tasks, such as writing documents (Word), playing games, or browsing the web.
Operating System (OS)
A suite of basic programs that acts as an interface between the user and the computer hardware. It manages memory, files, processes, and devices. Examples: Windows, Linux, macOS, Android.
Compiler
Translation software that converts a high-level programming language code into machine language (executable) code all at once.
Machine Language
The lowest-level programming language consisting of 0s and 1s, directly understood by the computer hardware.
assembly Language
Low-level programming language that uses mnemonics (codes) instead of machine code.
High-Level Language
Programming language that uses English-like words and is easier for humans to read and write (e.g., C++, Java, Python, BASIC).
Virus
A malicious program that replicates itself by modifying other programs/files. It requires a host program to spread.
Worm
A standalone malicious program that replicates itself over computer networks to consume resources. It does not need a host program.
Trojan Horse
A malicious program disguised as legitimate software. It does not replicate but opens a backdoor or causes damage when executed.
Antivirus Software
Programs designed to detect, prevent, and remove malicious software like viruses and worms.
CLI (Command Line Interface)
A text-based user interface where users type commands to interact with the OS (e.g., DOS, Linux Terminal).
GUI (Graphical User Interface)
A visual user interface that allows users to interact with the computer using graphical elements like icons, windows, and menus (e.g., Windows Desktop).
Motherboard
The main printed circuit board in a computer that connects all other components (CPU, Memory, Ports).
Power Supply Unit (PSU)
Component that converts mains AC electricity to low-voltage DC power for the internal components of a computer.
Ports
Physical docking points on a computer for connecting external devices (e.g., USB, HDMI, Audio jack).
Artificial Intelligence (AI)
The simulation of human intelligence processes by machines, especially computer systems. Feature of 5th Generation computers.
Supercomputer
The fastest and most powerful type of computer, used for complex scientific calculations.
Mainframe
A large, powerful computer used by large organizations for bulk data processing and critical applications.
Microcomputer
A small, relatively inexpensive computer with a microprocessor as its CPU. Includes Desktops, Laptops, Tablets.
