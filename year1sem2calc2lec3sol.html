INTEGRATION BY PARTS PRACTICE - LECTURE 3

Integration by Parts & Tabular Method Practice

Problem 1
\[ \int x\,e^x\,dx \]
Let \(u=x\), \(dv=e^x dx\); then \(du=dx\), \(v=e^x\)
\(xe^x - \int e^x dx\)
\[ = xe^x - e^x + C \]
Problem 2
\[ \int x^2\sin x\,dx \]
IBP twice: \(u=x^2\), \(dv=\sin x\,dx\)
First: \(-x^2\cos x+2\int x\cos x\,dx\)
Second IBP on \(\int x\cos x\,dx = x\sin x+\cos x\)
\[ = -x^2\cos x + 2x\sin x + 2\cos x + C \]
Problem 3
\[ \int \ln x\,dx \]
Let \(u=\ln x\), \(dv=dx\); then \(du=\frac{1}{x}dx\), \(v=x\)
\(x\ln x - \int 1\,dx\)
\[ = x\ln x - x + C \]
Problem 4
\[ \int e^x\sin x\,dx \]
IBP twice, cyclic case
\(I = e^x\sin x - e^x\cos x - I\)
\(2I = e^x(\sin x - \cos x)\)
\[ = \frac{e^x(\sin x - \cos x)}{2} + C \]
Problem 5
\[ \int x^3 e^{x^2}\,dx \]
Rewrite as \(\int x^2 \cdot x e^{x^2}dx\); let \(u=x^2\)
Becomes \(\frac{1}{2}\int u e^u du\); IBP: \(ue^u - e^u\)
\[ = \frac{1}{2}(x^2-1)e^{x^2}+C \]
