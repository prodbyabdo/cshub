CALCULUS 2 SOLUTIONS

DAY 3: INTEGRATION BY PARTS & INVERSE HYPERBOLIC

20 Step-by-Step Practice Problems

Problem 1
\[ \int x e^x \, dx \]
Use Integration by Parts. Choose \(u=x\) and \(dv=e^x \, dx\).
Then \(du=dx\) and \(v=e^x\).
Apply formula: \(uv - \int v \, du = x e^x - \int e^x \, dx\).
\[ x e^x - e^x + C \]
Problem 2
\[ \int x \cos x \, dx \]
Choose \(u=x\) (Algebraic) and \(dv=\cos x \, dx\) (Trig).
Then \(du=dx\) and \(v=\sin x\).
\(uv - \int v \, du = x \sin x - \int \sin x \, dx\).
\[ x \sin x + \cos x + C \]
Problem 3
\[ \int x^2 \ln x \, dx \]
By LIATE, choose \(u=\ln x\) and \(dv=x^2 \, dx\).
Then \(du=\frac{1}{x}dx\) and \(v=\frac{x^3}{3}\).
\(\frac{x^3}{3} \ln x - \int \frac{x^3}{3} \cdot \frac{1}{x} \, dx = \frac{x^3}{3} \ln x - \int \frac{x^2}{3} \, dx\).
\[ \frac{x^3}{3} \ln x - \frac{x^3}{9} + C \]
Problem 4
\[ \int \ln x \, dx \]
Choose \(u=\ln x\) and \(dv=dx\).
Then \(du=\frac{1}{x}dx\) and \(v=x\).
\(x \ln x - \int x \cdot \frac{1}{x} \, dx = x \ln x - \int 1 \, dx\).
\[ x \ln x - x + C \]
Problem 5
\[ \int x \sqrt{x+1} \, dx \]
Choose \(u=x\) and \(dv=(x+1)^{1/2} \, dx\).
Then \(du=dx\) and \(v=\frac{2}{3}(x+1)^{3/2}\).
\(\frac{2}{3}x(x+1)^{3/2} - \int \frac{2}{3}(x+1)^{3/2} \, dx\).
\[ \frac{2}{3}x(x+1)^{3/2} - \frac{4}{15}(x+1)^{5/2} + C \]
Problem 6
\[ I = \int e^x \sin x \, dx \]
Recurrent integration. Let \(u=e^x, dv=\sin x dx \implies du=e^xdx, v=-\cos x\).
\(I = -e^x\cos x + \int e^x\cos x \, dx\).
Second IBP on the new integral: let \(u=e^x, dv=\cos x dx \implies du=e^xdx, v=\sin x\).
\(\int e^x\cos x dx = e^x\sin x - \int e^x\sin x dx = e^x\sin x - I\).
Substitute back: \(I = -e^x\cos x + e^x\sin x - I \implies 2I = e^x(\sin x - \cos x)\).
\[ \frac{e^x(\sin x - \cos x)}{2} + C \]
Problem 7
\[ \int x^2 e^x \, dx \]
First IBP: \(u=x^2, dv=e^x dx \implies du=2xdx, v=e^x\).
\(x^2 e^x - \int 2x e^x dx\).
Second IBP on \(\int x e^x dx\): this evaluates to \(x e^x - e^x\) (from Prob 1).
\[ x^2 e^x - 2x e^x + 2e^x + C \]
Problem 8
\[ \int \sin^{-1} x \, dx \]
Choose \(u=\sin^{-1} x\) and \(dv=dx\).
Then \(du=\frac{1}{\sqrt{1-x^2}}dx\) and \(v=x\).
\(x \sin^{-1} x - \int \frac{x}{\sqrt{1-x^2}} dx\).
For the remaining integral, use substitution: \(w=1-x^2 \implies dw=-2xdx\).
\[ x \sin^{-1} x + \sqrt{1-x^2} + C \]
Problem 9
\[ \int \tan^{-1} x \, dx \]
Choose \(u=\tan^{-1} x\) and \(dv=dx\).
Then \(du=\frac{1}{1+x^2}dx\) and \(v=x\).
\(x \tan^{-1} x - \int \frac{x}{1+x^2} dx\).
Use natural log rule: numerator is half derivative of denominator (\(d/dx(1+x^2)=2x\)).
\[ x \tan^{-1} x - \frac{1}{2}\ln(1+x^2) + C \]
Problem 10
\[ \int \frac{\ln x}{x^2} \, dx \]
Rewrite as \(\int x^{-2} \ln x \, dx\).
Choose \(u=\ln x\) and \(dv=x^{-2}dx\).
Then \(du=\frac{1}{x}dx\) and \(v=-x^{-1} = -\frac{1}{x}\).
\(-\frac{\ln x}{x} - \int \left(-\frac{1}{x}\right)\left(\frac{1}{x}\right) dx = -\frac{\ln x}{x} + \int x^{-2} dx\).
\[ -\frac{\ln x}{x} - \frac{1}{x} + C \]
Problem 11
\[ \int e^{2x} \cos(3x) \, dx \]
Recurrent IBP. Let \(I = \int e^{2x} \cos(3x) \, dx\).
First IBP: \(u=e^{2x}, dv=\cos(3x)dx \implies du=2e^{2x}dx, v=\frac{1}{3}\sin(3x)\).
\(I = \frac{1}{3}e^{2x}\sin(3x) - \frac{2}{3}\int e^{2x}\sin(3x)dx\).
Second IBP: \(u=e^{2x}, dv=\sin(3x)dx \implies du=2e^{2x}dx, v=-\frac{1}{3}\cos(3x)\).
\(I = \frac{1}{3}e^{2x}\sin(3x) - \frac{2}{3}\left(-\frac{1}{3}e^{2x}\cos(3x) + \frac{2}{3}\int e^{2x}\cos(3x)dx\right)\).
\(I = \frac{1}{3}e^{2x}\sin(3x) + \frac{2}{9}e^{2x}\cos(3x) - \frac{4}{9}I\).
\(\frac{13}{9}I = e^{2x}\left(\frac{3\sin(3x) + 2\cos(3x)}{9}\right)\).
\[ \frac{e^{2x}(3\sin(3x) + 2\cos(3x))}{13} + C \]
Problem 12
\[ \int x \sec^2 x \, dx \]
Choose \(u=x\) and \(dv=\sec^2 x \, dx\).
Then \(du=dx\) and \(v=\tan x\).
\(x \tan x - \int \tan x \, dx\).
Recall \(\int \tan x \, dx = -\ln|\cos x| = \ln|\sec x|\).
\[ x \tan x + \ln|\cos x| + C \]
Problem 13
\[ \int x^3 e^{x^2} \, dx \]
Split the integral: \(\int x^2 (x e^{x^2}) \, dx\).
Choose \(u=x^2\) and \(dv=x e^{x^2} dx\).
Then \(du=2xdx\) and \(v=\frac{1}{2}e^{x^2}\) (via substitution \(w=x^2\)).
\(\frac{1}{2}x^2 e^{x^2} - \int x e^{x^2} dx\).
\[ \frac{1}{2}x^2 e^{x^2} - \frac{1}{2}e^{x^2} + C \]
Problem 14
\[ \int \frac{1}{\sqrt{x^2+9}} \, dx \]
This matches the standard inverse hyperbolic sine form: \(\int \frac{dx}{\sqrt{x^2+a^2}}\).
Here \(a=3\). The formula yields \(\sinh^{-1}(x/a)\) or \(\ln|x + \sqrt{x^2+a^2}|\).
\[ \sinh^{-1}\left(\frac{x}{3}\right) + C \quad \text{or} \quad \ln\left|x + \sqrt{x^2+9}\right| + C \]
Problem 15
\[ \int \frac{1}{\sqrt{x^2-16}} \, dx \]
This matches the standard inverse hyperbolic cosine form: \(\int \frac{dx}{\sqrt{x^2-a^2}}\).
Here \(a=4\). The formula yields \(\cosh^{-1}(x/a)\) or \(\ln|x + \sqrt{x^2-a^2}|\).
\[ \cosh^{-1}\left(\frac{x}{4}\right) + C \quad \text{or} \quad \ln\left|x + \sqrt{x^2-16}\right| + C \]
Problem 16
\[ \int \frac{1}{25-x^2} \, dx \]
This matches the inverse hyperbolic tangent form: \(\int \frac{dx}{a^2-x^2}\).
Here \(a=5\). The formula is \(\frac{1}{a}\tanh^{-1}(x/a)\).
\[ \frac{1}{5}\tanh^{-1}\left(\frac{x}{5}\right) + C \]
Problem 17
\[ \int \frac{x}{\sqrt{x^4+1}} \, dx \]
Rewrite as \(\int \frac{x}{\sqrt{(x^2)^2+1}} \, dx\). Let \(u=x^2 \implies du=2xdx\).
The integral becomes \(\frac{1}{2}\int \frac{du}{\sqrt{u^2+1}}\).
This is the \(\sinh^{-1}\) form with \(a=1\).
\[ \frac{1}{2}\sinh^{-1}(x^2) + C \]
Problem 18
\[ \int \frac{e^x}{\sqrt{e^{2x}-4}} \, dx \]
Rewrite as \(\int \frac{e^x}{\sqrt{(e^x)^2-4}} \, dx\). Let \(u=e^x \implies du=e^xdx\).
The integral becomes \(\int \frac{du}{\sqrt{u^2-4}}\).
This is the \(\cosh^{-1}\) form with \(a=2\).
\[ \cosh^{-1}\left(\frac{e^x}{2}\right) + C \]
Problem 19
\[ \int \ln(x^2+1) \, dx \]
IBP: let \(u=\ln(x^2+1)\) and \(dv=dx\).
Then \(du=\frac{2x}{x^2+1}dx\) and \(v=x\).
\(x\ln(x^2+1) - \int \frac{2x^2}{x^2+1} dx = x\ln(x^2+1) - 2\int \left(1 - \frac{1}{x^2+1}\right) dx\).
\[ x\ln(x^2+1) - 2x + 2\tan^{-1}(x) + C \]
Problem 20
\[ \int (\ln x)^2 \, dx \]
IBP: let \(u=(\ln x)^2\) and \(dv=dx\).
Then \(du=2(\ln x)\frac{1}{x}dx\) and \(v=x\).
\(x(\ln x)^2 - \int x \left( \frac{2\ln x}{x} \right) dx = x(\ln x)^2 - 2\int \ln x \, dx\).
From Problem 4, \(\int \ln x dx = x\ln x - x\).
\[ x(\ln x)^2 - 2x\ln x + 2x + C \]
