1. Integration by Parts

Derivation

From the product rule for differentiation:

\[\frac{d}{dx}(uv) = u\frac{dv}{dx} + v\frac{du}{dx}\]

Rearranging and integrating both sides:

\[u\,dv = d(uv) - v\,du\]

The IBP Formula:

\[\int u\,dv = uv - \int v\,du\]

2. LIATE Rule â€” Choosing u

To select \(u\), use the LIATE priority (first in list = best choice for \(u\)):

Priority Type Examples
1st Logarithmic \(\ln x,\;\log_a x\)
2nd Inverse Trig \(\sin^{-1}x,\;\tan^{-1}x\)
3rd Algebraic \(x^n,\;\sqrt{x}\)
4th Trigonometric \(\sin x,\;\cos x\)
5th Exponential \(e^x,\;a^x\)

3. Worked Examples â€” Integration by Parts

EXAMPLE 1

Evaluate \(\displaystyle\int xe^x\,dx\)

Let \(u = x \Rightarrow du = dx\), \(dv = e^x dx \Rightarrow v = e^x\):

\[\int xe^x\,dx = xe^x - \int e^x\,dx = xe^x - e^x + C = e^x(x-1) + C\]
EXAMPLE 2

Evaluate \(\displaystyle\int x\sin x\,dx\)

Let \(u = x \Rightarrow du = dx\), \(dv = \sin x\,dx \Rightarrow v = -\cos x\):

\[\int x\sin x\,dx = -x\cos x + \int \cos x\,dx = -x\cos x + \sin x + C\]
EXAMPLE 3

Evaluate \(\displaystyle\int x^2\ln x\,dx\)

Let \(u = \ln x \Rightarrow du = \frac{1}{x}dx\), \(dv = x^2\,dx \Rightarrow v = \frac{x^3}{3}\):

\[\int x^2\ln x\,dx = \frac{x^3}{3}\ln x - \int \frac{x^3}{3}\cdot\frac{1}{x}\,dx = \frac{x^3}{3}\ln x - \frac{1}{3}\int x^2\,dx\] \[= \frac{x^3}{3}\ln x - \frac{x^3}{9} + C\]
EXAMPLE 4

Evaluate \(\displaystyle\int \tan^{-1}x\,dx\)

Let \(u = \tan^{-1}x \Rightarrow du = \frac{1}{x^2+1}dx\), \(dv = dx \Rightarrow v = x\):

\[\int \tan^{-1}x\,dx = x\tan^{-1}x - \int \frac{x}{x^2+1}\,dx = x\tan^{-1}x - \frac{1}{2}\ln(x^2+1) + C\]
EXAMPLE 5

Evaluate \(\displaystyle\int \ln x\,dx\)

Let \(u = \ln x \Rightarrow du = \frac{1}{x}dx\), \(dv = dx \Rightarrow v = x\):

\[\int \ln x\,dx = x\ln x - \int x\cdot\frac{1}{x}\,dx = x\ln x - x + C\]
EXAMPLE 6

Evaluate \(\displaystyle\int \sqrt{x^2+a^2}\,dx\)

Let \(u = \sqrt{x^2+a^2}\), \(du = \frac{x}{\sqrt{x^2+a^2}}dx\), \(dv = dx \Rightarrow v = x\):

\[I = x\sqrt{x^2+a^2} - \int \frac{x^2}{\sqrt{x^2+a^2}}\,dx\]

Rewrite \(x^2 = (x^2+a^2) - a^2\):

\[I = x\sqrt{x^2+a^2} - I + a^2\int \frac{dx}{\sqrt{x^2+a^2}}\] \[2I = x\sqrt{x^2+a^2} + a^2\sinh^{-1}\!\left(\frac{x}{a}\right)\] \[I = \frac{1}{2}x\sqrt{x^2+a^2} + \frac{a^2}{2}\sinh^{-1}\!\left(\frac{x}{a}\right) + C\]
EXAMPLE 7 â€” CYCLIC / RECURRING

Evaluate \(\displaystyle I = \int e^{3x}\cos 2x\,dx\)

First application: \(u = e^{3x}\), \(dv = \cos 2x\,dx \Rightarrow v = \frac{1}{2}\sin 2x\):

\[I = \frac{1}{2}e^{3x}\sin 2x - \frac{3}{2}\int e^{3x}\sin 2x\,dx\]

Second application on the remaining integral:

\[I = \frac{1}{2}e^{3x}\sin 2x + \frac{3}{4}e^{3x}\cos 2x - \frac{9}{4}I\]

Solve for \(I\):

\[\frac{13}{4}I = \frac{1}{2}e^{3x}\sin 2x + \frac{3}{4}e^{3x}\cos 2x\] \[I = \frac{e^{3x}}{13}(2\sin 2x + 3\cos 2x) + C\]

4. Tabular Method (Repeated IBP)

The tabular method is a convenient way to organise repeated integration by parts. It works when one factor differentiates to zero in finitely many steps.

Procedure:

  1. Create two columns: D (differentiate) and I (integrate).
  2. Column D: write \(u\) and differentiate repeatedly until \(0\).
  3. Column I: write \(dv\) and integrate repeatedly (no \(+C\)).
  4. Multiply diagonally with alternating signs \(+,\,-,\,+,\,-,\ldots\)
  5. Sum all diagonal products and add \(+C\).

Example â€” Tabular: \(\int x^2 e^x\,dx\)

EXAMPLE 8
Sign Differentiate \(f(x)\) Integrate \(g(x)\)
\(+\) \(x^2\) \(e^x\)
\(-\) \(2x\) \(e^x\)
\(+\) \(2\) \(e^x\)
\(0\)
\[\int x^2 e^x\,dx = x^2 e^x - 2xe^x + 2e^x + C = e^x(x^2 - 2x + 2) + C\]

Example â€” Tabular: \(\int x^2\cos(nx)\,dx\)

EXAMPLE 9

Where \(n\) is a positive integer:

Sign Differentiate Integrate
\(+\) \(x^2\) \(\frac{1}{n}\sin(nx)\)
\(-\) \(2x\) \(-\frac{1}{n^2}\cos(nx)\)
\(+\) \(2\) \(-\frac{1}{n^3}\sin(nx)\)
\(0\)
\[\int x^2\cos(nx)\,dx = \frac{x^2}{n}\sin(nx) + \frac{2x}{n^2}\cos(nx) - \frac{2}{n^3}\sin(nx) + C\]

5. Recurring Integrals

For integrands like \(e^x\sin x\) or \(e^x\cos x\), two applications of IBP return the original integral \(I\). Solve algebraically:

\[I = [\text{expression}] - kI \implies I = \frac{[\text{expression}]}{1 + k}\]
EXAMPLE 10

Evaluate \(\displaystyle I = \int e^x\sin x\,dx\)

First IBP: \(u = e^x\), \(dv = \sin x\,dx \Rightarrow v = -\cos x\):

\[I = -e^x\cos x + \int e^x\cos x\,dx\]

Second IBP on \(\int e^x\cos x\,dx\):

\[\int e^x\cos x\,dx = e^x\sin x - \int e^x\sin x\,dx = e^x\sin x - I\]

Substitute back:

\[I = -e^x\cos x + e^x\sin x - I\] \[2I = e^x(\sin x - \cos x)\]
\[\int e^x\sin x\,dx = \frac{e^x}{2}(\sin x - \cos x) + C\]

  • For \(\ln x\), \(\tan^{-1}x\), \(\sin^{-1}x\): always set them as \(u\) with \(dv = dx\).
  • If one factor is a polynomial, tabular method is fastest.

8. Six Standard Integrals

These six integrals arise frequently and are derived using trigonometric or hyperbolic substitutions. Memorise the results; understand the derivations.

# Integral Result
1 \(\displaystyle\int \frac{dx}{\sqrt{a^2-x^2}}\) \(\sin^{-1}\!\left(\frac{x}{a}\right) + C\)
2 \(\displaystyle\int \sqrt{a^2-x^2}\,dx\) \(\frac{a^2}{2}\sin^{-1}\!\left(\frac{x}{a}\right) + \frac{x}{2}\sqrt{a^2-x^2} + C\)
3 \(\displaystyle\int \frac{dx}{\sqrt{a^2+x^2}}\) \(\sinh^{-1}\!\left(\frac{x}{a}\right) + C\)
4 \(\displaystyle\int \sqrt{a^2+x^2}\,dx\) \(\frac{x}{2}\sqrt{a^2+x^2} + \frac{a^2}{2}\sinh^{-1}\!\left(\frac{x}{a}\right) + C\)
5 \(\displaystyle\int \frac{dx}{\sqrt{x^2-a^2}}\) \(\cosh^{-1}\!\left(\frac{x}{a}\right) + C\)
6 \(\displaystyle\int \sqrt{x^2-a^2}\,dx\) \(\frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\cosh^{-1}\!\left(\frac{x}{a}\right) + C\)

9. \(\displaystyle\int \frac{dx}{\sqrt{a^2-x^2}}\)

SUBSTITUTION: \(x = a\sin\theta\)

Then \(dx = a\cos\theta\,d\theta\) and \(\sqrt{a^2 - a^2\sin^2\theta} = a\cos\theta\):

\[\int \frac{a\cos\theta\,d\theta}{a\cos\theta} = \int d\theta = \theta + C\]

Since \(\theta = \sin^{-1}\!\left(\frac{x}{a}\right)\):

\[\int \frac{dx}{\sqrt{a^2-x^2}} = \sin^{-1}\!\left(\frac{x}{a}\right) + C\]

10. \(\displaystyle\int \sqrt{a^2-x^2}\,dx\)

SUBSTITUTION: \(x = a\sin\theta\)

\(dx = a\cos\theta\,d\theta\), \(\sqrt{a^2-a^2\sin^2\theta} = a\cos\theta\):

\[\int a\cos\theta \cdot a\cos\theta\,d\theta = a^2\int\cos^2\theta\,d\theta\]

Use \(\cos^2\theta = \frac{1+\cos 2\theta}{2}\):

\[= \frac{a^2}{2}\left(\theta + \frac{1}{2}\sin 2\theta\right) + C = \frac{a^2}{2}\theta + \frac{a^2}{2}\sin\theta\cos\theta + C\]

Back-substitute: \(\sin\theta = \frac{x}{a}\), \(\cos\theta = \frac{\sqrt{a^2-x^2}}{a}\):

\[\int \sqrt{a^2-x^2}\,dx = \frac{a^2}{2}\sin^{-1}\!\left(\frac{x}{a}\right) + \frac{x}{2}\sqrt{a^2-x^2} + C\]

11. \(\displaystyle\int \frac{dx}{\sqrt{a^2+x^2}}\)

SUBSTITUTION: \(x = a\sinh\theta\)

\(dx = a\cosh\theta\,d\theta\) and \(\sqrt{a^2+a^2\sinh^2\theta} = a\cosh\theta\):

\[\int \frac{a\cosh\theta\,d\theta}{a\cosh\theta} = \int d\theta = \theta + C\]

Since \(\theta = \sinh^{-1}\!\left(\frac{x}{a}\right)\):

\[\int \frac{dx}{\sqrt{a^2+x^2}} = \sinh^{-1}\!\left(\frac{x}{a}\right) + C\]

12. \(\displaystyle\int \sqrt{a^2+x^2}\,dx\)

SUBSTITUTION: \(x = a\sinh\theta\)

\(dx = a\cosh\theta\,d\theta\), using \(\cosh^2\theta - \sinh^2\theta = 1\):

\[\int a\cosh\theta \cdot a\cosh\theta\,d\theta = a^2\int\cosh^2\theta\,d\theta\]

Use \(\cosh^2\theta = \frac{\cosh 2\theta + 1}{2}\):

\[= \frac{a^2}{2}\left(\frac{\sinh 2\theta}{2} + \theta\right) + C = \frac{a^2}{2}\sinh\theta\cosh\theta + \frac{a^2}{2}\theta + C\]

Back-substitute: \(\sinh\theta = \frac{x}{a}\), \(\cosh\theta = \sqrt{1+\frac{x^2}{a^2}}\):

\[\int \sqrt{a^2+x^2}\,dx = \frac{x}{2}\sqrt{a^2+x^2} + \frac{a^2}{2}\sinh^{-1}\!\left(\frac{x}{a}\right) + C\]

13. \(\displaystyle\int \frac{dx}{\sqrt{x^2-a^2}}\)

SUBSTITUTION: \(x = a\cosh\theta\)

\(dx = a\sinh\theta\,d\theta\) and \(\sqrt{a^2\cosh^2\theta - a^2} = a\sinh\theta\):

\[\int \frac{a\sinh\theta\,d\theta}{a\sinh\theta} = \int d\theta = \theta + C\]

Since \(\theta = \cosh^{-1}\!\left(\frac{x}{a}\right)\):

\[\int \frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}\!\left(\frac{x}{a}\right) + C\]

14. \(\displaystyle\int \sqrt{x^2-a^2}\,dx\)

SUBSTITUTION: \(x = a\cosh\theta\)

\(dx = a\sinh\theta\,d\theta\), \(\sqrt{a^2\cosh^2\theta - a^2} = a\sinh\theta\):

\[a^2\int\sinh^2\theta\,d\theta\]

Use \(\sinh^2\theta = \frac{\cosh 2\theta - 1}{2}\):

\[= \frac{a^2}{2}\left(\frac{\sinh 2\theta}{2} - \theta\right) + C = \frac{a^2}{2}\sinh\theta\cosh\theta - \frac{a^2}{2}\theta + C\]

Back-substitute: \(\cosh\theta = \frac{x}{a}\), \(\sinh\theta = \frac{\sqrt{x^2-a^2}}{a}\):

\[\int \sqrt{x^2-a^2}\,dx = \frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\cosh^{-1}\!\left(\frac{x}{a}\right) + C\]

15. Quick Reference â€” Substitution Table

Expression Substitution Identity Used
\(\sqrt{a^2-x^2}\) \(x = a\sin\theta\) \(1-\sin^2\theta = \cos^2\theta\)
\(\sqrt{a^2+x^2}\) \(x = a\sinh\theta\) \(1+\sinh^2\theta = \cosh^2\theta\)
\(\sqrt{x^2-a^2}\) \(x = a\cosh\theta\) \(\cosh^2\theta - 1 = \sinh^2\theta\)
ðŸ’¡ Day 5 Tips:
  • These six results are standard â€” memorise them and recognise the radical form to pick the right substitution.
  • \(\sqrt{a^2-x^2}\) â†’ trig sub (\(\sin\theta\)); \(\sqrt{a^2+x^2}\) and \(\sqrt{x^2-a^2}\) â†’ hyperbolic sub (\(\sinh\theta\) or \(\cosh\theta\)).
  • Always convert back to \(x\) using \(\sin\theta = x/a\), \(\sinh\theta = x/a\), or \(\cosh\theta = x/a\).
  • For the square root integrals (2, 4, 6), use the double-angle identity to integrate the squared trig/hyperbolic function.
  • Don't forget the Jacobian factor from \(dx\) â€” it cancels in the simple cases but contributes \(a^2\) in the squared cases.
