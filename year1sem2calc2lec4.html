1. Six Standard Integrals

These six standard forms come up frequently in special integration problems. Memorize the results and the matching substitution.

#IntegralResult
1\(\displaystyle\int \frac{dx}{\sqrt{a^2-x^2}}\)\(\sin^{-1}\!\left(\frac{x}{a}\right) + C\)
2\(\displaystyle\int \sqrt{a^2-x^2}\,dx\)\(\frac{a^2}{2}\sin^{-1}\!\left(\frac{x}{a}\right) + \frac{x}{2}\sqrt{a^2-x^2} + C\)
3\(\displaystyle\int \frac{dx}{\sqrt{a^2+x^2}}\)\(\sinh^{-1}\!\left(\frac{x}{a}\right) + C\)
4\(\displaystyle\int \sqrt{a^2+x^2}\,dx\)\(\frac{x}{2}\sqrt{a^2+x^2} + \frac{a^2}{2}\sinh^{-1}\!\left(\frac{x}{a}\right) + C\)
5\(\displaystyle\int \frac{dx}{\sqrt{x^2-a^2}}\)\(\cosh^{-1}\!\left(\frac{x}{a}\right) + C\)
6\(\displaystyle\int \sqrt{x^2-a^2}\,dx\)\(\frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\cosh^{-1}\!\left(\frac{x}{a}\right) + C\)

2. \(\displaystyle\int \frac{dx}{\sqrt{a^2-x^2}}\)

SUBSTITUTION: \(x = a\sin\theta\)
\[\int \frac{a\cos\theta\,d\theta}{a\cos\theta} = \int d\theta = \theta + C\]

Since \(\theta = \sin^{-1}\!\left(\frac{x}{a}\right)\):

\[\int \frac{dx}{\sqrt{a^2-x^2}} = \sin^{-1}\!\left(\frac{x}{a}\right) + C\]

3. \(\displaystyle\int \sqrt{a^2-x^2}\,dx\)

SUBSTITUTION: \(x = a\sin\theta\)
\[\int a\cos\theta \cdot a\cos\theta\,d\theta = a^2\int\cos^2\theta\,d\theta\] \[\int \sqrt{a^2-x^2}\,dx = \frac{a^2}{2}\sin^{-1}\!\left(\frac{x}{a}\right) + \frac{x}{2}\sqrt{a^2-x^2} + C\]

4. \(\displaystyle\int \frac{dx}{\sqrt{a^2+x^2}}\)

SUBSTITUTION: \(x = a\sinh\theta\)
\[\int \frac{a\cosh\theta\,d\theta}{a\cosh\theta} = \int d\theta = \theta + C\] \[\int \frac{dx}{\sqrt{a^2+x^2}} = \sinh^{-1}\!\left(\frac{x}{a}\right) + C\]

5. \(\displaystyle\int \sqrt{a^2+x^2}\,dx\)

SUBSTITUTION: \(x = a\sinh\theta\)
\[\int a\cosh\theta \cdot a\cosh\theta\,d\theta = a^2\int\cosh^2\theta\,d\theta\] \[\int \sqrt{a^2+x^2}\,dx = \frac{x}{2}\sqrt{a^2+x^2} + \frac{a^2}{2}\sinh^{-1}\!\left(\frac{x}{a}\right) + C\]

6. \(\displaystyle\int \frac{dx}{\sqrt{x^2-a^2}}\)

SUBSTITUTION: \(x = a\cosh\theta\)
\[\int \frac{a\sinh\theta\,d\theta}{a\sinh\theta} = \int d\theta = \theta + C\] \[\int \frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}\!\left(\frac{x}{a}\right) + C\]

7. \(\displaystyle\int \sqrt{x^2-a^2}\,dx\)

SUBSTITUTION: \(x = a\cosh\theta\)
\[a^2\int\sinh^2\theta\,d\theta\] \[\int \sqrt{x^2-a^2}\,dx = \frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\cosh^{-1}\!\left(\frac{x}{a}\right) + C\]

8. Quick Reference - Substitution Table

ExpressionSubstitutionIdentity Used
\(\sqrt{a^2-x^2}\)\(x = a\sin\theta\)\(1-\sin^2\theta = \cos^2\theta\)
\(\sqrt{a^2+x^2}\)\(x = a\sinh\theta\)\(1+\sinh^2\theta = \cosh^2\theta\)
\(\sqrt{x^2-a^2}\)\(x = a\cosh\theta\)\(\cosh^2\theta - 1 = \sinh^2\theta\)
Tips:
  • Recognize the radical form first, then pick the matching trig or hyperbolic substitution.
  • The denominator forms usually collapse immediately after substitution.
  • The square-root forms require integrating a squared trig or hyperbolic function.
