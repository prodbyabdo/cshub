1. Integration by Substitution (u-Substitution)

If an integrand has the form \(f(g(x))\cdot g'(x)\), we set \(u = g(x)\) so that \(du = g'(x)\,dx\) and the integral simplifies:

\[\int f(g(x))\,g'(x)\,dx = \int f(u)\,du\]

Procedure

  1. Identify an inner function \(u = g(x)\).
  2. Compute \(du = g'(x)\,dx\).
  3. Rewrite every \(x\)-expression in terms of \(u\) and \(du\).
  4. Integrate with respect to \(u\).
  5. Substitute back to express the result in terms of \(x\).

Worked Examples

EXAMPLE A

Find \(\displaystyle\int \frac{2x-9}{\sqrt{x^2-9x+1}}\,dx\)

Step 1 — let \(u = x^2 - 9x + 1\), then \(du = (2x-9)\,dx\):

\[\int \frac{du}{\sqrt{u}} = \int u^{-1/2}\,du = 2u^{1/2} + c = 2\sqrt{x^2 - 9x + 1} + c\]
EXAMPLE B

Find \(\displaystyle\int 3x^2\cos(x^3)\,dx\)

Let \(u = x^3\), then \(du = 3x^2\,dx\):

\[\int \cos(u)\,du = \sin(u) + c = \sin(x^3) + c\]
EXAMPLE C — ADJUSTING CONSTANTS

Find \(\displaystyle\int x\,e^{x^2}\,dx\)

Let \(u = x^2\), then \(du = 2x\,dx\), so \(x\,dx = \tfrac{1}{2}\,du\):

\[\frac{1}{2}\int e^{u}\,du = \frac{1}{2}e^{u} + c = \frac{1}{2}e^{x^2} + c\]

2. Definite Integrals by Substitution

When using substitution in a definite integral, change the limits from \(x\)-values to \(u\)-values:

\[\int_{a}^{b} f(g(x))\,g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du\]

This avoids needing to substitute back to \(x\).

EXAMPLE — DEFINITE

Evaluate \(\displaystyle\int_{0}^{2} x(x^2+1)^3\,dx\)

Let \(u = x^2+1\), \(du = 2x\,dx\). When \(x=0\): \(u=1\); when \(x=2\): \(u=5\).

\[\frac{1}{2}\int_{1}^{5} u^3\,du = \frac{1}{2}\left[\frac{u^4}{4}\right]_{1}^{5} = \frac{1}{8}(625 - 1) = 78\]

3. Completing the Square in Integration

When the integrand contains a quadratic \(ax^2 + bx + c\) under a square root or in a denominator, completing the square rewrites it into a standard form that matches known integral formulas.

Standard Forms Unlocked

\[\int \frac{dx}{\sqrt{a^2 - u^2}} = \sin^{-1}\!\left(\frac{u}{a}\right) + c\] \[\int \frac{dx}{a^2 + u^2} = \frac{1}{a}\tan^{-1}\!\left(\frac{u}{a}\right) + c\] \[\int \frac{dx}{u\sqrt{u^2 - a^2}} = \frac{1}{a}\sec^{-1}\!\left|\frac{u}{a}\right| + c\]

Worked Example

EXAMPLE — COMPLETING THE SQUARE

Find \(\displaystyle\int \frac{dx}{\sqrt{8x - x^2}}\)

Step 1 — complete the square:

\[8x - x^2 = -(x^2 - 8x) = -(x^2 - 8x + 16) + 16 = 16 - (x-4)^2\]

Step 2 — recognise the \(\sin^{-1}\) form with \(a=4\), \(u=x-4\):

\[\int \frac{dx}{\sqrt{16 - (x-4)^2}} = \sin^{-1}\!\left(\frac{x-4}{4}\right) + c\]
EXAMPLE — QUADRATIC DENOMINATOR

Find \(\displaystyle\int \frac{dx}{x^2 + 6x + 13}\)

Step 1 — complete the square: \(x^2+6x+13 = (x+3)^2 + 4\)

Step 2 — use \(\tan^{-1}\) form with \(a=2\), \(u=x+3\):

\[\int \frac{dx}{(x+3)^2 + 4} = \frac{1}{2}\tan^{-1}\!\left(\frac{x+3}{2}\right) + c\]

4. Common Substitution Patterns

Integrand Pattern Substitution Result
\(\displaystyle\int \frac{f'(x)}{f(x)}\,dx\) \(u = f(x)\) \(\ln|f(x)| + c\)
\(\displaystyle\int f'(x)\,[f(x)]^n\,dx\) \(u = f(x)\) \(\dfrac{[f(x)]^{n+1}}{n+1} + c\)
\(\displaystyle\int f'(x)\,e^{f(x)}\,dx\) \(u = f(x)\) \(e^{f(x)} + c\)
\(\displaystyle\int \frac{dx}{\sqrt{a^2-u^2}}\) Complete the square \(\sin^{-1}(u/a) + c\)
\(\displaystyle\int \frac{dx}{a^2+u^2}\) Complete the square \(\frac{1}{a}\tan^{-1}(u/a) + c\)

5. Examination-Style Questions

QUESTION 1

Find \(\displaystyle\int \frac{6x^2}{(x^3+4)^2}\,dx\).

Let \(u = x^3+4\), \(du = 3x^2\,dx\):

\[\frac{6}{3}\int u^{-2}\,du = 2\cdot\frac{u^{-1}}{-1} + c = -\frac{2}{x^3+4} + c\]
QUESTION 2

Evaluate \(\displaystyle\int_{0}^{1} \frac{x}{\sqrt{1+x^2}}\,dx\).

Let \(u = 1+x^2\), \(du = 2x\,dx\). Limits: \(u(0)=1,\; u(1)=2\).

\[\frac{1}{2}\int_{1}^{2} u^{-1/2}\,du = \frac{1}{2}\left[2u^{1/2}\right]_{1}^{2} = \sqrt{2} - 1\]
QUESTION 3

Find \(\displaystyle\int \frac{\cos x}{1+\sin^2 x}\,dx\).

Let \(u = \sin x\), \(du = \cos x\,dx\):

\[\int \frac{du}{1+u^2} = \tan^{-1}(u) + c = \tan^{-1}(\sin x) + c\]
QUESTION 4

Find \(\displaystyle\int \frac{dx}{\sqrt{3-2x-x^2}}\).

Complete the square: \(3-2x-x^2 = -(x^2+2x-3) = -(x^2+2x+1)+4 = 4-(x+1)^2\).

\[\int \frac{dx}{\sqrt{4-(x+1)^2}} = \sin^{-1}\!\left(\frac{x+1}{2}\right) + c\]
QUESTION 5

Find \(\displaystyle\int (1+\tan x)^2\sec^2 x\,dx\).

Let \(u = 1+\tan x\), \(du = \sec^2 x\,dx\):

\[\int u^2\,du = \frac{u^3}{3} + c = \frac{(1+\tan x)^3}{3} + c\]

6. Quick Reference — Lecture 4 Rules

💡 Lecture 4 Tips:
  • Always look for the derivative of the inner function lurking in the integrand.
  • For definite integrals, change the limits when substituting — saves time.
  • If a quadratic appears under a root or in a denominator, try completing the square first.
  • Adjust constants: if \(du = 3x^2\,dx\) but you only have \(x^2\,dx\), multiply/divide by 3.
  • Check your answer by differentiating — the chain rule should return the integrand.
