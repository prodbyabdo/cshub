QUESTION 01 · U-SUBSTITUTION

Evaluate: \(\displaystyle\int x\,(x^2+1)^6\,dx\)

Let \(u=x^2+1,\quad du=2x\,dx \Rightarrow x\,dx=\tfrac{1}{2}du\)

\[\tfrac{1}{2}\int u^6\,du = \frac{u^7}{14}+C\]
\[\dfrac{(x^2+1)^7}{14}+C\]

QUESTION 27 · IBP CYCLIC

Evaluate: \(\displaystyle\int e^{3x}\cos2x\,dx\)

Apply IBP twice (cyclic). Let \(I=\int e^{3x}\cos2x\,dx\).

\(u=e^{3x},\;dv=\cos2x\,dx\): \(I=\tfrac{e^{3x}\sin2x}{2}-\tfrac{3}{2}\int e^{3x}\sin2x\,dx\)

Second IBP on \(\int e^{3x}\sin2x\,dx\): \(-\tfrac{e^{3x}\cos2x}{2}+\tfrac{3}{2}\int e^{3x}\cos2x\,dx\)

Substitute back: \(I=\tfrac{e^{3x}\sin2x}{2}+\tfrac{3e^{3x}\cos2x}{4}-\tfrac{9}{4}I \Rightarrow \tfrac{13}{4}I=\cdots\)

\[\dfrac{e^{3x}(3\cos2x+2\sin2x)}{13}+C\]

QUESTION 49 · IBP

Evaluate: \(\displaystyle\int e^x \sin x\,dx\)

Cyclic IBP. Let \(I = \int e^x \sin x\,dx\). \(u=\sin x, \; dv=e^x\,dx \Rightarrow du=\cos x\,dx, \; v=e^x\)

\[I = e^x\sin x - \int e^x\cos x\,dx\]

Second IBP: \(u=\cos x, \; dv=e^x\,dx \Rightarrow du=-\sin x\,dx, \; v=e^x\)

\[I = e^x\sin x - \left(e^x\cos x - \int e^x(-\sin x)\,dx\right) = e^x(\sin x - \cos x) - I\] \[2I = e^x(\sin x - \cos x) \Rightarrow I = \frac{e^x}{2}(\sin x - \cos x)\]
\[\dfrac{e^x}{2}(\sin x - \cos x) + C\]

\[\dfrac{x}{4\sqrt{x^2+4}} + C\]

Evaluate: \(\displaystyle\int \sqrt{1 + e^x}\,dx\)

\(A=1/4, \; B=-1/4, \; C=0, \; D=-1/2\)

\[\int \left( \frac{1/4}{x-1} - \frac{1/4}{x+1} - \frac{1/2}{x^2+1} \right) dx\]
\[\dfrac{1}{4}\ln\left|\dfrac{x-1}{x+1}\right| - \dfrac{1}{2}\arctan x + C\]

\[x\ln(x^2+1) - 2x + 2\arctan x + C\]
QUESTION 60 · DEFINITE INTEGRAL (HARD)

Evaluate: \(\displaystyle\int_0^{\pi/4} \frac{\sin x + \cos x}{3 + \sin 2x}\,dx\)

Let \(u = \sin x - \cos x \Rightarrow du = (\cos x + \sin x)dx\).

\(u^2 = \sin^2 x - 2\sin x\cos x + \cos^2 x = 1 - \sin 2x \Rightarrow \sin 2x = 1 - u^2\)

Denominator: \(3 + \sin 2x = 3 + (1 - u^2) = 4 - u^2\)

Limits: \(x=0 \to u=-1\); \(x=\pi/4 \to u=0\)

\[\int_{-1}^0 \frac{du}{4-u^2} = \frac{1}{4} \left[ \ln\left|\frac{2+u}{2-u}\right| \right]_{-1}^0 = \frac{1}{4}(\ln 1 - \ln(1/3))\]
