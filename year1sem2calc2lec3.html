1. Integration by Parts

From the product rule for differentiation:

\[\frac{d}{dx}(uv) = u\frac{dv}{dx} + v\frac{du}{dx}\]

Rearranging and integrating gives:

\[\int u\,dv = uv - \int v\,du\]

Main formula:

\[\int u\,dv = uv - \int v\,du\]

2. LIATE Rule

Choose \(u\) using the LIATE order:

PriorityTypeExamples
1stLogarithmic\(\ln x,\;\log_a x\)
2ndInverse trig\(\sin^{-1}x,\;\tan^{-1}x\)
3rdAlgebraic\(x^n,\;\sqrt{x}\)
4thTrigonometric\(\sin x,\;\cos x\)
5thExponential\(e^x,\;a^x\)

3. Worked Examples

EXAMPLE 1

Evaluate \(\displaystyle\int xe^x\,dx\)

\[\int xe^x\,dx = xe^x - \int e^x\,dx = e^x(x-1) + C\]
EXAMPLE 2

Evaluate \(\displaystyle\int x\sin x\,dx\)

\[\int x\sin x\,dx = -x\cos x + \int \cos x\,dx = -x\cos x + \sin x + C\]
EXAMPLE 3

Evaluate \(\displaystyle\int x^2\ln x\,dx\)

\[\int x^2\ln x\,dx = \frac{x^3}{3}\ln x - \frac{1}{3}\int x^2\,dx = \frac{x^3}{3}\ln x - \frac{x^3}{9} + C\]
EXAMPLE 4

Evaluate \(\displaystyle\int \tan^{-1}x\,dx\)

\[\int \tan^{-1}x\,dx = x\tan^{-1}x - \int \frac{x}{x^2+1}\,dx = x\tan^{-1}x - \frac{1}{2}\ln(x^2+1) + C\]
EXAMPLE 5

Evaluate \(\displaystyle\int \ln x\,dx\)

\[\int \ln x\,dx = x\ln x - \int 1\,dx = x\ln x - x + C\]

4. Recurrent Integration

Some integrals return to the original integral after two applications of integration by parts. Treat the original integral as \(I\) and solve algebraically.

EXAMPLE 6

Evaluate \(\displaystyle I = \int e^x\sin x\,dx\)

First IBP:

\[I = -e^x\cos x + \int e^x\cos x\,dx\]

Second IBP:

\[\int e^x\cos x\,dx = e^x\sin x - I\]

Substitute back:

\[I = -e^x\cos x + e^x\sin x - I\] \[2I = e^x(\sin x - \cos x)\] \[\int e^x\sin x\,dx = \frac{e^x}{2}(\sin x - \cos x) + C\]

5. Tabular Method

The tabular method is a shortcut for repeated integration by parts when one factor differentiates to zero after a finite number of steps.

Procedure:

  1. Write a differentiation column and an integration column.
  2. Differentiate the polynomial factor until it becomes zero.
  3. Integrate the other factor repeatedly.
  4. Multiply diagonally with alternating signs.
EXAMPLE 7

Evaluate \(\displaystyle\int x^2 e^x\,dx\)

SignDifferentiateIntegrate
\(+\)\(x^2\)\(e^x\)
\(-\)\(2x\)\(e^x\)
\(+\)\(2\)\(e^x\)
\(0\)
\[\int x^2 e^x\,dx = x^2e^x - 2xe^x + 2e^x + C\]
EXAMPLE 8

Evaluate \(\displaystyle\int x^2\cos(nx)\,dx\)

SignDifferentiateIntegrate
\(+\)\(x^2\)\(\frac{1}{n}\sin(nx)\)
\(-\)\(2x\)\(-\frac{1}{n^2}\cos(nx)\)
\(+\)\(2\)\(-\frac{1}{n^3}\sin(nx)\)
\(0\)
\[\int x^2\cos(nx)\,dx = \frac{x^2}{n}\sin(nx) + \frac{2x}{n^2}\cos(nx) - \frac{2}{n^3}\sin(nx) + C\]
