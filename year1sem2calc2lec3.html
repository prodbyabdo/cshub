1. Integration by Parts

Based on the product rule for differentiation, Integration by Parts is used to integrate the product of two functions:

\[\int u \, dv = uv - \int v \, du\]

How to choose \(u\) and \(dv\) (LIATE Rule)

Choose \(u\) to be the function that comes first in this list (and \(dv\) will be the rest of the integrand):

  1. Logarithmic functions (e.g., \(\ln x\))
  2. Inverse trigonometric functions (e.g., \(\tan^{-1} x\))
  3. Algebraic functions (e.g., \(x^2\), \(3x\))
  4. Trigonometric functions (e.g., \(\sin x\), \(\cos x\))
  5. Exponential functions (e.g., \(e^x\))

Worked Examples

EXAMPLE A — ALGEBRAIC & EXPONENTIAL

Find \(\displaystyle\int x e^x \, dx\)

Step 1 — Choose \(u\) and \(dv\):

Let \(u = x\) (Algebraic) \(\implies du = dx\)

Let \(dv = e^x \, dx\) (Exponential) \(\implies v = e^x\)

Step 2 — Apply formula:

\[\int x e^x \, dx = x e^x - \int e^x \, dx = x e^x - e^x + C\]
EXAMPLE B — LOGARITHMIC

Find \(\displaystyle\int \ln x \, dx\)

Step 1 — Choose \(u\) and \(dv\):

Let \(u = \ln x\) (Logarithmic) \(\implies du = \frac{1}{x} dx\)

Let \(dv = dx\) (Algebraic) \(\implies v = x\)

Step 2 — Apply formula:

\[\int \ln x \, dx = x \ln x - \int x \left(\frac{1}{x}\right) dx = x \ln x - \int 1 \, dx = x \ln x - x + C\]
EXAMPLE C — INVERSE TRIGONOMETRIC

Find \(\displaystyle\int \tan^{-1} x \, dx\)

Let \(u = \tan^{-1} x \implies du = \frac{1}{1+x^2} dx\)

Let \(dv = dx \implies v = x\)

\[= x \tan^{-1} x - \int \frac{x}{1+x^2} dx\]

For the remaining integral, use substitution (Natural Log Rule) with \(w = 1+x^2\), \(dw = 2x\,dx\):

\[= x \tan^{-1} x - \frac{1}{2} \ln(1+x^2) + C\]

2. Recurrent Integration by Parts

Sometimes, applying Integration by Parts twice returns the original integral. When this happens, we treat the integral as a variable (say, \(I\)) and solve for it algebraically.

EXAMPLE — EXPONENTIAL & TRIG

Find \(I = \displaystyle\int e^x \cos x \, dx\)

First IBP: Let \(u = e^x\), \(dv = \cos x \, dx \implies du = e^x \, dx\), \(v = \sin x\)

\[I = e^x \sin x - \int e^x \sin x \, dx\]

Second IBP on \(\int e^x \sin x \, dx\): Let \(u = e^x\), \(dv = \sin x \, dx \implies du = e^x \, dx\), \(v = -\cos x\)

\[\int e^x \sin x \, dx = -e^x \cos x - \int (-e^x \cos x) \, dx = -e^x \cos x + I\]

Substitute back:

\[I = e^x \sin x - (-e^x \cos x + I) = e^x \sin x + e^x \cos x - I\] \[2I = e^x(\sin x + \cos x) \implies I = \frac{e^x(\sin x + \cos x)}{2} + C\]

3. Inverse Hyperbolic Functions

Similar to standard inverse trigonometric functions, inverse hyperbolic functions appear from specific radical denominators.

Integrand Resulting Inverse Hyperbolic
\(\displaystyle\int \frac{dx}{\sqrt{x^2 + a^2}}\) \(\sinh^{-1}\left(\frac{x}{a}\right) + C \quad \text{or} \quad \ln|x + \sqrt{x^2+a^2}| + C\)
\(\displaystyle\int \frac{dx}{\sqrt{x^2 - a^2}}\) \(\cosh^{-1}\left(\frac{x}{a}\right) + C \quad \text{or} \quad \ln|x + \sqrt{x^2-a^2}| + C\)
\(\displaystyle\int \frac{dx}{a^2 - x^2}\) \(\frac{1}{a}\tanh^{-1}\left(\frac{x}{a}\right) + C \quad \text{for } |x| < a\)
💡 Note on Logarithmic Forms:

Inverse hyperbolic functions can always be expressed in terms of natural logarithms. Many exams will accept either form, but logarithmic form is often preferred if bounds are given to evaluate.

4. Quick Reference — Lecture 3 Rules

Method Formula / Rule
Integration by Parts \(\int u \, dv = uv - \int v \, du\)
LIATE Rule for \(u\) Logarithmic, Inverse Trig, Algebraic, Trigonometric, Exponential
Recurrent Integration If \(I\) appears on the RHS, add it to the LHS and divide.
Inverse \(\sinh\) form \(\int \frac{1}{\sqrt{x^2+a^2}} dx = \sinh^{-1}(x/a) + C\)
Inverse \(\cosh\) form \(\int \frac{1}{\sqrt{x^2-a^2}} dx = \cosh^{-1}(x/a) + C\)
