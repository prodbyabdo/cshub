1. Basic Integration

Definition of the Indefinite Integral

\[\int f(x)\,dx = F(x) + C \quad \text{where } F'(x) = f(x)\]

The constant \(C\) is the constant of integration and must always be included.

Fundamental Theorem of Calculus

Part 1:

\[\frac{d}{dx}\int_a^x f(t)\,dt = f(x)\]

Part 2:

\[\int_a^b f(x)\,dx = F(b) - F(a)\]

2. Integration by Substitution

The Rule:

\[\int f(g(x))\,g'(x)\,dx = \int f(u)\,du \quad \text{where } u = g(x),\;\; du = g'(x)\,dx\]

Procedure

  1. Choose \(u = g(x)\) — typically the inner function or the expression under a root or power.
  2. Compute \(du = g'(x)\,dx\), then isolate \(dx\).
  3. Rewrite the entire integral in terms of \(u\).
  4. Integrate with respect to \(u\).
  5. Back-substitute \(u = g(x)\) to express the answer in \(x\).

Common Substitution Patterns

Integrand contains Try substitution
\((ax+b)^n\) \(u = ax+b\)
\(e^{g(x)} \cdot g'(x)\) \(u = g(x)\)
\(\dfrac{g'(x)}{g(x)}\) \(u = g(x)\)
\(\sin^n x \cos x\) \(u = \sin x\)
\(\cos^n x \sin x\) \(u = \cos x\)
\(f(\sqrt{x})\) \(u = \sqrt{x},\;\; x = u^2\)

Definite Integrals — Change of Limits

When limits are in \(x\), convert them to \(u\):

\[\int_a^b f(g(x))\,g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du\]

3. Integration by Parts

\[\int u\,dv = uv - \int v\,du\]

Equivalent form:

\[\int f(x)\,g'(x)\,dx = f(x)\,g(x) - \int g(x)\,f'(x)\,dx\]

Choosing u and dv — LIATE Rule

Priority Type Examples
1st — choose as \(u\) Logarithmic \(\ln x,\;\log_a x\)
2nd — choose as \(u\) Inverse Trig \(\sin^{-1}x,\;\tan^{-1}x\)
3rd — choose as \(u\) Algebraic \(x^n,\;\sqrt{x}\)
4th — choose as \(u\) Trigonometric \(\sin x,\;\cos x\)
5th — choose as \(u\) Exponential \(e^x,\;a^x\)

Cyclic Case

For integrands like \(e^x \sin x\) or \(e^x \cos x\), two applications of parts return the original integral \(I\). Solve algebraically:

\[I = [\text{expression}] - kI \implies I = \frac{[\text{expression}]}{1 + k}\]

4. Tabular Method

A fast version of repeated Integration by Parts. Works when one factor differentiates to zero in finitely many steps.

Procedure

  1. Create two columns: D (differentiate) and I (integrate).
  2. Column D: write \(u\) and differentiate repeatedly until \(0\).
  3. Column I: write \(dv\) and integrate repeatedly (no \(+C\)).
  4. Multiply diagonally with alternating signs \(+,\,-,\,+,\,-,\ldots\)
  5. Sum all diagonal products and add \(+C\).

Sign Pattern

Sign D column I column
\(+\) \(u\) \(v_1\)
\(-\) \(u'\) \(v_2\)
\(+\) \(u''\) \(v_3\)
\(-\) \(u'''\) \(v_4\)
\(\vdots\) \(\vdots\) \(\vdots\)
\(0\)

Result:

\[\int u\,dv = (+)\,u\,v_1 - u'\,v_2 + u''\,v_3 - u'''\,v_4 + \cdots + C\]

5. Special Integration — Trigonometric Integrals

Powers of Sine and Cosine

Case 1 — odd power of \(\cos x\):
Peel off one \(\cos x\), use \(\cos^2 x = 1 - \sin^2 x\), let \(u = \sin x\).

Case 2 — odd power of \(\sin x\):
Peel off one \(\sin x\), use \(\sin^2 x = 1 - \cos^2 x\), let \(u = \cos x\).

Case 3 — both powers even:
Use the half-angle identities below.

Essential Identities

\[\sin^2 x = \frac{1 - \cos 2x}{2} \qquad \cos^2 x = \frac{1 + \cos 2x}{2}\] \[\sin x \cos x = \frac{\sin 2x}{2}\] \[\sin^2 x + \cos^2 x = 1 \qquad 1 + \tan^2 x = \sec^2 x \qquad 1 + \cot^2 x = \csc^2 x\]

Powers of Tangent and Secant

Form Strategy
\(\int \tan^n x\,dx\) Use \(\tan^2 x = \sec^2 x - 1\), reduce
\(\int \sec^n x\,dx\) (\(n\) even) Factor \(\sec^2 x\), use \(\sec^2 x = 1+\tan^2 x\), let \(u=\tan x\)
\(\int \sec^n x\,dx\) (\(n\) odd) Integration by parts or reduction formula
\(\int \tan^m x \sec^n x\,dx\) (\(n\) even) Let \(u = \tan x\)
\(\int \tan^m x \sec^n x\,dx\) (\(m\) odd) Let \(u = \sec x\)

Product-to-Sum Formulas

\[\sin A \cos B = \tfrac{1}{2}[\sin(A-B) + \sin(A+B)]\] \[\sin A \sin B = \tfrac{1}{2}[\cos(A-B) - \cos(A+B)]\] \[\cos A \cos B = \tfrac{1}{2}[\cos(A-B) + \cos(A+B)]\]

6. Fraction Integration — Partial Fractions

Used for \(\displaystyle\int \dfrac{P(x)}{Q(x)}\,dx\) where \(\deg P < \deg Q\).
If \(\deg P \geq \deg Q\): perform polynomial long division first.

Decomposition Cases

Factor in \(Q(x)\) Partial Fraction Terms
Linear: \((ax+b)\) \(\dfrac{A}{ax+b}\)
Repeated linear: \((ax+b)^n\) \(\dfrac{A_1}{ax+b} + \dfrac{A_2}{(ax+b)^2} + \cdots + \dfrac{A_n}{(ax+b)^n}\)
Irreducible quadratic: \((ax^2+bx+c)\) \(\dfrac{Ax+B}{ax^2+bx+c}\)
Repeated quadratic: \((ax^2+bx+c)^n\) \(\dfrac{A_1x+B_1}{ax^2+bx+c} + \cdots + \dfrac{A_nx+B_n}{(ax^2+bx+c)^n}\)

Key Results

\[\int \frac{A}{ax+b}\,dx = \frac{A}{a}\ln|ax+b| + C\] \[\int \frac{Ax+B}{x^2+a^2}\,dx = \frac{A}{2}\ln(x^2+a^2) + \frac{B}{a}\tan^{-1}\!\left(\frac{x}{a}\right) + C\]

Procedure

  1. Factor \(Q(x)\) completely over the reals.
  2. Write the partial fraction decomposition for each factor type.
  3. Multiply through by \(Q(x)\) and expand both sides.
  4. Solve for constants by substituting convenient values of \(x\) or comparing coefficients.
  5. Integrate each partial fraction term separately.

7. Trigonometric Substitution

Used to eliminate radicals of the forms \(\sqrt{a^2-x^2}\), \(\sqrt{a^2+x^2}\), and \(\sqrt{x^2-a^2}\).

Substitution Table

Expression Substitution Identity Used Simplified
\(\sqrt{a^2 - x^2}\) \(x = a\sin\theta,\quad dx = a\cos\theta\,d\theta\) \(1 - \sin^2\theta = \cos^2\theta\) \(a\cos\theta\)
\(\sqrt{a^2 + x^2}\) \(x = a\tan\theta,\quad dx = a\sec^2\theta\,d\theta\) \(1 + \tan^2\theta = \sec^2\theta\) \(a\sec\theta\)
\(\sqrt{x^2 - a^2}\) \(x = a\sec\theta,\quad dx = a\sec\theta\tan\theta\,d\theta\) \(\sec^2\theta - 1 = \tan^2\theta\) \(|a\tan\theta|\)

Reference Triangles for Back-Substitution

\(x = a\sin\theta\):   opp \(= x\), hyp \(= a\), adj \(= \sqrt{a^2-x^2}\)

\(x = a\tan\theta\):   opp \(= x\), adj \(= a\), hyp \(= \sqrt{a^2+x^2}\)

\(x = a\sec\theta\):   hyp \(= x\), adj \(= a\), opp \(= \sqrt{x^2-a^2}\)

8. Clever Substitution

Rationalizing Substitution

For integrands with \(\sqrt[n]{ax+b}\), let \(u = \sqrt[n]{ax+b}\):
Then \(x = \dfrac{u^n - b}{a}\) and \(dx = \dfrac{n\,u^{n-1}}{a}\,du\).

For mixed powers like \(x^{1/2}\) and \(x^{1/3}\), let \(u = x^{1/6}\) (LCM of denominators).

Weierstrass (Half-Angle) Substitution

For rational functions of \(\sin x\) and \(\cos x\), let \(t = \tan\!\left(\dfrac{x}{2}\right)\):

\[\sin x = \frac{2t}{1+t^2}, \qquad \cos x = \frac{1-t^2}{1+t^2}, \qquad dx = \frac{2}{1+t^2}\,dt\]

Logarithmic Pattern Recognition

\[\int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + C\]

Recognising this pattern avoids a full substitution in many cases.

9. Properties of Trigonometric Integrals

Even and Odd Symmetry

If \(f\) is even \(\;[\,f(-x) = f(x)\,]\):

\[\int_{-a}^{a} f(x)\,dx = 2\int_0^a f(x)\,dx\]

If \(f\) is odd \(\;[\,f(-x) = -f(x)\,]\):

\[\int_{-a}^{a} f(x)\,dx = 0\]

Wallis's Formulas

\[\int_0^{\pi/2} \sin^n x\,dx = \int_0^{\pi/2} \cos^n x\,dx\]

n even:   \(\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2}\)   e.g. \(n=4: \dfrac{3\cdot1}{4\cdot2}\cdot\dfrac{\pi}{2} = \dfrac{3\pi}{16}\)

n odd:   \(\dfrac{(n-1)!!}{n!!}\)   e.g. \(n=3: \dfrac{2}{3}\)

Reduction Formulas

Integral Reduction Formula
\(\int \sin^n x\,dx\) \(-\dfrac{\sin^{n-1}x\cos x}{n} + \dfrac{n-1}{n}\displaystyle\int \sin^{n-2}x\,dx\)
\(\int \cos^n x\,dx\) \(\dfrac{\cos^{n-1}x\sin x}{n} + \dfrac{n-1}{n}\displaystyle\int \cos^{n-2}x\,dx\)
\(\int \tan^n x\,dx\) \(\dfrac{\tan^{n-1}x}{n-1} - \displaystyle\int \tan^{n-2}x\,dx\)
\(\int \sec^n x\,dx\) \(\dfrac{\sec^{n-2}x\tan x}{n-1} + \dfrac{n-2}{n-1}\displaystyle\int \sec^{n-2}x\,dx\)

10. Areas Between Curves

If \(f(x) \geq g(x)\) on \([a,b]\):

\[A = \int_a^b \bigl[f(x) - g(x)\bigr]\,dx\]

General (curves may cross):

\[A = \int_a^b \left|f(x) - g(x)\right|\,dx\]

Integrating with Respect to y

When curves are expressed as \(x = f(y)\) and \(x = g(y)\) with \(f(y) \geq g(y)\) on \([c,d]\):

\[A = \int_c^d \bigl[f(y) - g(y)\bigr]\,dy\]

Procedure

  1. Sketch the curves to identify which is on top.
  2. Find intersection points by solving \(f(x) = g(x)\) — these are the limits.
  3. If curves cross within the interval, split the integral at each crossing.
  4. Evaluate and sum all areas (use absolute values throughout).

11. Multiple Integrals

Double Integral

\[\iint_R f(x,y)\,dA = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x,y)\,dy\,dx\]

Fubini's Theorem

If \(f\) is continuous on the rectangle \(R = [a,b]\times[c,d]\):

\[\iint_R f(x,y)\,dA = \int_a^b\int_c^d f(x,y)\,dy\,dx = \int_c^d\int_a^b f(x,y)\,dx\,dy\]

Properties

\[\iint_R [f \pm g]\,dA = \iint_R f\,dA \pm \iint_R g\,dA\] \[\iint_R k\,f\,dA = k\iint_R f\,dA\] \[\iint_R 1\,dA = \text{Area of } R\]

Polar Coordinates

\(x = r\cos\theta,\quad y = r\sin\theta,\quad dA = r\,dr\,d\theta\)

\[\iint_R f(x,y)\,dA = \int_\alpha^\beta \int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta,\, r\sin\theta)\; r\,dr\,d\theta\]

The extra \(r\) is the Jacobian — never omit it.

12. Triple Integrals

\[\iiint_E f(x,y,z)\,dV = \int_a^b \int_{g_1(x)}^{g_2(x)} \int_{h_1(x,y)}^{h_2(x,y)} f(x,y,z)\,dz\,dy\,dx\]
\[\iiint_E 1\,dV = \text{Volume of } E\]

Cylindrical Coordinates

\(x = r\cos\theta,\quad y = r\sin\theta,\quad z = z\)

\[\iiint_E f\,dV = \iiint f(r\cos\theta,\, r\sin\theta,\, z)\; r\,dz\,dr\,d\theta\]

Spherical Coordinates

\(x = \rho\sin\phi\cos\theta,\quad y = \rho\sin\phi\sin\theta,\quad z = \rho\cos\phi\)

\[\iiint_E f\,dV = \iiint f\;\rho^2\sin\phi\,d\rho\,d\phi\,d\theta\]

Jacobian: \(\rho^2\sin\phi\)

💡 Study Tips:
  • Always write \(+C\) for indefinite integrals — it costs marks if missing.
  • For substitution: find the piece whose derivative also appears in the integrand.
  • LIATE tells you which factor to call \(u\) in Integration by Parts.
  • Partial fractions require \(\deg(\text{num}) < \deg(\text{denom})\) — divide first if not.
  • Trig substitution: identify the radical type, draw the reference triangle to back-substitute.
  • For areas: always sketch and check which curve is above the other.
  • Double and triple integrals: evaluate from the inside out, one variable at a time.
