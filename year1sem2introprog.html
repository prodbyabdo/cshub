§1 · Java Platform — JDK, JRE, JVM & Compilation

What is Java?

Java is a high-level, object-oriented language first released in 1995. It runs across many platforms through the JVM.

The Three Layers

  • JDK: Java Development Kit (compiler javac, debugger, libraries, and JRE).
  • JRE: Java Runtime Environment (JVM + libraries needed to run programs).
  • JVM: Java Virtual Machine that executes bytecode on each OS.

Compilation Pipeline

Main.java → javac → Main.class → java Main → output

Java compiles source code to bytecode, then JVM interprets or JIT-compiles it at runtime.

Why Java?

  • Platform-independent (write once, run anywhere).
  • Popular and in-demand.
  • Object-oriented and structured.
  • Secure, fast, and has a huge ecosystem.

Java Editions

  • Java SE: Standard Edition (used in this course).
  • Java EE: Enterprise web systems.
  • Java ME: Mobile/embedded systems.
  • JavaFX: UI applications.

§3 · Hello World & Program Structure
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
  • Class name must match filename (Main.java).
  • main() is the entry point.
  • Statements end with ;.
  • Code blocks use curly braces { }.
§4 · Output
System.out.println("Hello World!");
System.out.print("Same line ");
System.out.print("output");
System.out.println(3 + 3);

Use double quotes for text literals. println adds a new line; print does not.

§7 · Type Casting

Widening: automatic small → large type (int to double).

Narrowing: manual large → small type (double to int), possible data loss.

int myInt = 9;
double myDouble = myInt;

double d = 9.78;
int n = (int) d; // 9

  • Logical: &&, ||, !.

§10 · Conditions & Branching
if (x > 18) {
    System.out.println("x is greater than 18");
} else if (x == 18) {
    System.out.println("x is 18");
} else {
    System.out.println("x is less than 18");
}
switch (day) {
    case 1: dayName = "Monday"; break;
    case 2: dayName = "Tuesday"; break;
    default: dayName = "Invalid";
}
§11 · Loops
while (i < 5) { i++; }

do { i++; } while (i < 5);

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
for (int i = 0; i < 10; i++) {
    if (i == 4) break;
}

for (int i = 0; i < 10; i++) {
    if (i == 4) continue;
}
