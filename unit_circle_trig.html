Unit Circle

Visual Trigonometry & Angle Systems

The Unit Circle

The unit circle is a circle with a radius of $1$ centered at the origin $(0,0)$. For any angle $\theta$ measured counterclockwise from the positive x-axis:

$$ x = \cos(\theta) $$ $$ y = \sin(\theta) $$

A full rotation is $360^\circ$. Angles greater than $360^\circ$ wrap around, and negative angles move clockwise.

Coordinates are always given as $(x, y)$ which corresponds to $(\cos \theta, \sin \theta)$.

Quadrant Signs

Memory trick: "All Students Take Calculus" (ASTC) tells you which functions are positive in each quadrant.

Quadrant II ($90^\circ - 180^\circ$)
(-, +)
Sine is positive
Quadrant I ($0^\circ - 90^\circ$)
(+, +)
All are positive
Quadrant III ($180^\circ - 270^\circ$)
(-, -)
Tangent is positive
Quadrant IV ($270^\circ - 360^\circ$)
(+, -)
Cosine is positive

Reference Angles

A reference angle ($\alpha$) is the acute angle ($0^\circ$ to $90^\circ$) formed between the terminal side and the x-axis.

Case 1: $\theta = 150^\circ$

Quadrant II. Reference is $180^\circ - 150^\circ = 30^\circ$.

$\sin(150^\circ) = \sin(30^\circ) = 0.5$

Case 2: $\theta = 330^\circ$

Quadrant IV. Reference is $360^\circ - 330^\circ = 30^\circ$.

$\sin(330^\circ) = -\sin(30^\circ) = -0.5$

Notice how both have the same magnitude (0.5) because they share the same reference angle, but different signs based on their quadrant.

Angle Families

Angles with the same reference angle share the same absolute value of sine (and cosine). They form a "family".

30° Family
30°, 150°, 210°, 330°
All have $|\sin(\theta)| = 0.5$
45° Family
45°, 135°, 225°, 315°
All have $|\sin(\theta)| = \frac{\sqrt{2}}{2}$
60° Family
60°, 120°, 240°, 300°
All have $|\sin(\theta)| = \frac{\sqrt{3}}{2}$

Symmetry Rules

These rules imply symmetry across the y-axis or origin on the unit circle:

$$ \sin(180^\circ - \theta) = \sin(\theta) \quad (\text{Quad II same as Quad I}) $$ $$ \sin(180^\circ + \theta) = -\sin(\theta) \quad (\text{Quad III opposite}) $$ $$ \sin(360^\circ - \theta) = -\sin(\theta) \quad (\text{Quad IV opposite}) $$

Geometrically, reflecting a point $(x,y)$ across the y-axis keeps $y$ the same ($180 - \theta$). Reflecting through the origin flips the sign of $y$ ($180 + \theta$).

Worked Examples

To find the sine of any angle, follow these steps: Reference $\to$ Quadrant $\to$ Sign $\to$ Value.

Example 1: $\sin(135^\circ)$
  • Reference: Ends in Quad II. $180^\circ - 135^\circ = 45^\circ$.
  • Quadrant: II ($90^\circ$ to $180^\circ$). Sine is Positive (+).
  • Value: $\sin(45^\circ) = \frac{\sqrt{2}}{2}$.
  • Result: $\sin(135^\circ) = +\frac{\sqrt{2}}{2}$.
Example 2: $\sin(210^\circ)$
  • Reference: Ends in Quad III. $210^\circ - 180^\circ = 30^\circ$.
  • Quadrant: III ($180^\circ$ to $270^\circ$). Sine is Negative (-).
  • Value: $\sin(30^\circ) = 0.5$.
  • Result: $\sin(210^\circ) = -0.5$.
Example 3: $\sin(330^\circ)$
  • Reference: Ends in Quad IV. $360^\circ - 330^\circ = 30^\circ$.
  • Quadrant: IV. Sine is Negative (-).
  • Value: $\sin(30^\circ) = 0.5$.
  • Result: $\sin(330^\circ) = -0.5$.

Quick Check Summary

  1. Find Reference Angle: angle with x-axis ($0-90^\circ$).
  2. Find Quadrant: determines the sign ($+,-$).
  3. Apply Sign: based on quadrant (ASTC rule).
  4. Use Base Value: from the common values table.
