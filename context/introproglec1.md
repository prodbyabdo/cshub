## Computer Hardware

```
雒ｶ Computer hardware is the collection of physical elements that constitutes a
computer system.
```
```
雒ｶ Computer hardware is various devices such as the keyboard, screen, mouse, disks,
memory, CD-ROM and processing units.
```
```

1. Input Unit

```
雒ｶ Input unit allow the user to enter information into the system, or control its
operation.
```
```
雒ｶ This unit obtains information from various input devices and places this
information at the disposal of the other units so that the information may be
processed.
```
```
雒ｶ Most personal computers have a mouse and keyboard, but laptop systems
typically use a touchpad instead of a mouse. Other input devices include
webcams, microphones, joysticks, and image scanners.
```
2. Output Unit

```
雒ｶ This unit takes information that has been processed by the computer and places it
on various output devices to make information available for use outside the
computer.
```
```
雒ｶ Most output from computer today is displayed on screens, printed on paper, or
used to control other devices.

```
```
難ｼ ROM (Read Only Memory) contains fundamental instructions that cannot be

4. Arithmetic and Logic Unit (ALU)

```
雒ｶ The ALU performs all the arithmetic and logic operations.
雒ｶ Example: addition, subtraction, comparison, etc.
```
5. Central Processing Unit (CPU)

```
雒ｶ The unit supervises the overall operation of the computer.
雒ｶ CPU
難ｼ Tells the input unit when information should be read into the memory unit.
難ｼ Tell the ALU when information from the memory should be used in
calculations.
難ｼ Tells the output unit when to send information from the memory unit to
certain output devices.
```
