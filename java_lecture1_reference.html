§1 Java Platform — JDK, JRE, JVM & Compilation

What is Java?

Java is a high-level, object-oriented language released by Sun Microsystems in 1995, now owned by Oracle. It runs on Windows, macOS, Linux, and many other platforms. Over 3 billion devices run Java today.

The Three Layers

JDK
Java Development Kit. Everything a developer needs: compiler (javac), debugger, libraries, and the JRE. You install this to write and compile Java programs.
JRE
Java Runtime Environment. The part needed to run already-compiled Java programs. Includes the JVM + standard libraries. End-users install this; developers install the JDK (which contains it).
JVM
Java Virtual Machine. An abstract machine that reads bytecode and executes it on your actual hardware. This is why Java is platform-independent — bytecode is the same everywhere; the JVM adapts it per OS.

The Compilation Pipeline

Unlike C++ (which compiles directly to machine code), Java compiles to an intermediate format called bytecode, then the JVM interprets/JIT-compiles bytecode at runtime.

Source CodeMain.java
→
Compilerjavac
→
BytecodeMain.class
→
JVMjava Main
→
Outputon any OS
C++ vs Java compilation In C++: g++ main.cpp -o main → produces a binary for that specific OS/CPU.
In Java: javac Main.java → produces Main.class (bytecode). Run anywhere with java Main.

§3 Hello World & Program Structure

Every Java program lives inside a class. The class name must match the filename exactly. Execution begins inside main().

Main.java — Hello World

Lecture Example · Hello World
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
Output
Hello World
public class Main
Declares a class named Main. File must be saved as Main.java. Java is case-sensitive.
public static void main
The entry point. Every Java program requires this exact signature. static means it belongs to the class itself, not an instance.
String[] args
Command-line arguments passed in. Required syntax even if unused.
System.out.println()
System is a built-in class. out is the output stream. println prints a line and adds a newline character.
Semicolon ;
Every statement ends with ; — same as C++.
Curly braces {}
Mark the start and end of code blocks — same as C++.

§7 Type Casting

Type casting converts a value from one type to another. Java has two kinds:

Widening
Smaller → larger type. Happens automatically. No data loss. byte → short → char → int → long → float → double
Narrowing
Larger → smaller type. Must be done manually. Risk of data loss.

Widening casting — automatic

Lecture Example · Widening Casting
int myInt = 9;
double myDouble = myInt; // automatic: int → double
System.out.println(myInt);    // 9
System.out.println(myDouble); // 9.0

Narrowing casting — manual

Lecture Example · Narrowing Casting
double myDouble = 9.78d;
int myInt = (int) myDouble; // manual: double → int (truncates)
System.out.println(myDouble); // 9.78
System.out.println(myInt);    // 9

Real-life — percentage score

Lecture Example · Score Percentage
int score = 90;
int maxScore = 100;
float percentage = (float) score / maxScore * 100;
System.out.println("Your score is: " + percentage + "%");
// Output: Your score is: 90.0%

§9 Strings & String Methods

In Java, String is an object (not a primitive). It has built-in methods you can call on it using dot notation.

String length

Lecture Example · String Length
String txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
System.out.println("The length of the txt string is: " + txt.length());
// Output: The length of the txt string is: 26

toUpperCase / toLowerCase

Lecture Example · Case Methods
String txt = "Hello World";
System.out.println(txt.toUpperCase()); // HELLO WORLD
System.out.println(txt.toLowerCase()); // hello world

String concatenation — + operator

Lecture Example · Concatenation
String firstName = "John";
String lastName = "Doe";
System.out.println(firstName + " " + lastName); // John Doe

concat() method

Lecture Example · concat()
String firstName = "John ";
String lastName = "Doe";
System.out.println(firstName.concat(lastName)); // John Doe
Common String methods summary s.length() — character count  |  s.toUpperCase() — all caps  |  s.toLowerCase() — all lower  |  s.charAt(i) — character at index i  |  s.equals("x") — compare content (not ==)  |  s.indexOf("x") — find position of substring

§10 Conditions & Branching

Simple if

Lecture Example · if Statement
int x = 20;
if (x > 18) {
    System.out.println("x is greater than 18");
}

if-else

Lecture Example · if-else
int time = 20;
if (time < 18) {
    System.out.println("Good day.");
} else {
    System.out.println("Good evening.");
}
// Output: Good evening.

else if

Lecture Example · else if
int time = 22;
if (time < 10) {
    System.out.println("Good morning.");
} else if (time < 18) {
    System.out.println("Good day.");
} else {
    System.out.println("Good evening.");
}
// Output: Good evening.

Real-life example — door code

Lecture Example · Door Code
int doorCode = 1337;
if (doorCode == 1337) {
    System.out.println("Correct code. The door is now open.");
} else {
    System.out.println("Wrong code. The door remains closed.");
}

Switch statement — weekday

Lecture Example · Switch
int day = 4;
String dayName;
switch (day) {
    case 1:  dayName = "Monday";    break;
    case 2:  dayName = "Tuesday";   break;
    case 3:  dayName = "Wednesday"; break;
    case 4:  dayName = "Thursday";  break;
    case 5:  dayName = "Friday";    break;
    case 6:  dayName = "Saturday";  break;
    case 7:  dayName = "Sunday";    break;
    default: dayName = "Invalid";
}
System.out.println(dayName); // Thursday
Don't forget break! Without break, Java "falls through" into the next case and keeps executing. Always end each case with break; unless fall-through is intentional.

§11 Loops

while loop

Lecture Example · while
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
// Output: 0 1 2 3 4 (each on new line)

do-while loop

Lecture Example · do-while
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
// Runs at least once — even if i >= 5 from the start

for loop — 0 to 4

Lecture Example · for loop
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
// Output: 0 1 2 3 4

for loop — even numbers 0–10

Lecture Example · Even Numbers
for (int i = 0; i <= 10; i += 2) {
    System.out.println(i);
}
// Output: 0 2 4 6 8 10

Nested loops

Lecture Example · Nested Loops
// Outer loop
for (int i = 1; i <= 2; i++) {
    System.out.println("Outer: " + i);
    // Inner loop
    for (int j = 1; j <= 3; j++) {
        System.out.println("  Inner: " + j);
    }
}
Output
Outer: 1   Inner: 1   Inner: 2   Inner: 3   Outer: 2   Inner: 1   Inner: 2   Inner: 3

break — exit loop early

Lecture Example · break
for (int i = 0; i < 10; i++) {
    if (i == 4) {
        break; // stop the loop when i equals 4
    }
    System.out.println(i);
}
// Output: 0 1 2 3

continue — skip one iteration

Lecture Example · continue
for (int i = 0; i < 10; i++) {
    if (i == 4) {
        continue; // skip 4, keep going
    }
    System.out.println(i);
}
// Output: 0 1 2 3 5 6 7 8 9

Real-life countdown

Lecture Example · Countdown
int countdown = 5;
while (countdown > 0) {
    System.out.println(countdown);
    countdown--;
}
System.out.println("Go!");
// Output: 5 4 3 2 1 Go!
