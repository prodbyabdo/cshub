TRIGONOMETRIC INTEGRALS PRACTICE - LECTURE 4

Trigonometric Integrals Practice

Problem 1
\[ \int \sin^3 x\,dx \]
\(\sin^3 x = \sin x(1-\cos^2 x)\)
Let \(u=\cos x\): \(-\int(1-u^2)du\)
\[ = -\cos x + \frac{\cos^3 x}{3} + C \]
Problem 2
\[ \int \cos^2 x\,dx \]
Half-angle: \(\cos^2 x=\frac{1+\cos 2x}{2}\)
\[ = \frac{x}{2}+\frac{\sin 2x}{4}+C \]
Problem 3
\[ \int \sin^2 x\cos^2 x\,dx \]
\(\sin^2 x\cos^2 x = \frac{1}{4}\sin^2 2x = \frac{1}{8}(1-\cos 4x)\)
\[ = \frac{x}{8}-\frac{\sin 4x}{32}+C \]
Problem 4
\[ \int \tan^3 x\sec x\,dx \]
\(\tan^3 x\sec x = (\sec^2 x-1)\sec x\tan x\)
Let \(u=\sec x\): \(\int(u^2-1)du\)
\[ = \frac{\sec^3 x}{3}-\sec x+C \]
Problem 5
\[ \int \sec^3 x\,dx \]
IBP + reduction: \(I=\sec x\tan x - \int \sec x\tan^2 x\,dx\)
\(I = \sec x\tan x - I + \int \sec x\,dx\)
\(2I = \sec x\tan x + \ln|\sec x+\tan x|\)
\[ = \frac{1}{2}\sec x\tan x + \frac{1}{2}\ln|\sec x+\tan x|+C \]
