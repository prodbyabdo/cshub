# Introduction to Computer Science - Definitions

**Computer Program**
A set of instructions written in a specific programming language which a computer can execute to solve a given problem.

**Programmer**
A person who analyzes problems and writes the source code for programs. Also called a developer, coder, or software engineer.

**Programming Language**
A formal language designed to communicate instructions to computers (e.g., C, C++, Java, Python).

**Source Code**
The code written by a programmer in a programming language.

**Algorithm**
A sequence of well-defined steps or instructions required to solve a problem.

**Flowchart**
A graphical representation of the steps to be followed to solve a problem using standard symbols.

**Pseudocode**
A tool that uses natural language-like statements to describe the logic of an algorithm. It is easier for humans to read than actual code.

**Compiler**
Software that converts programs written in a high-level language into machine language so the computer can execute them.

**Problem-Solving Phase**
The first phase of programming, dealing with understanding the problem, defining inputs/outputs, and deriving a solution (algorithm).

**Implementation Phase**
The second phase of programming, involving writing the code in a programming language and testing it.

**Terminal Symbol**
A flowchart symbol (oval) that denotes the beginning or end of an algorithm.

**Input/Output Symbol**
A flowchart symbol (parallelogram) that denotes inputting data or displaying output.

**Process Symbol**
A flowchart symbol (rectangle) that denotes a calculation or operation.

**Decision Symbol**
A flowchart symbol (diamond) used to represent a choice or branch (True/False, Yes/No).

**Flow Lines**
Arrows in a flowchart that indicate the direction of the logic flow.

**CPU (Central Processing Unit)**
The main processing unit of the computer, responsible for analyzing data and executing instructions.

**CU (Control Unit)**
Component of the CPU that directs the operation of the processor.

**ALU (Arithmetic and Logic Unit)**
Component of the CPU responsible for performing arithmetic (math) and logical operations.

**Variable**
A memory location used to store data that can change during program execution.

**Loop / Repetition**
A sequence of instructions that repeats while a certain condition is met (e.g., While loop, For loop).

**Selection**
A control structure that expects the computer to choose between two or more alternative actions (IF...THEN...ELSE).

**LAN (Local Area Network)**
A network that covers a small area such as an office or building.

**HTML (HyperText Markup Language)**
The technology used to create and structure web pages.

**WWW (World Wide Web)**
A collection of websites accessed through the Internet.
