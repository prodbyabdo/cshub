> 6. A computer can repeat a group of actions
> • When there is a sequence of processing steps that need to be repeated while a condition is met.
> • While loop: executes as long as the loop specified with the loop is true. When the condition becomes
> false the loop breaks. The condition test is at the top of the iterative statements.
> • Do .. While loop: similar to while loop except that the condition test is at the end of the iterative
> statements. This means that the Do .. While loop will execute at least one time.
> • For loop: consists of the following three parts:
> Initialization expression (i=0)
> Evaluation expression (i<5)
> Increment/decrement expression (i++)
