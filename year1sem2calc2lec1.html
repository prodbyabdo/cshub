1. Reversing the Process of Differentiation

To differentiate \(y = x^n\) we multiply by the power and reduce the power by one:

\[x^n \xrightarrow{\times n,\; \text{power} - 1} nx^{n-1}\]

Reversing this: given \(\dfrac{dy}{dx} = x^n\), find \(y\).

\[x^n \xrightarrow{\div (n+1),\; \text{power} + 1} \frac{x^{n+1}}{n+1}\]

Why We Need + C

If \(\dfrac{dy}{dx} = 6x^2\), adding 1 to the power and dividing by the new power gives:

\[y = \frac{6x^3}{3} = 2x^3\]

But differentiating \(2x^3 + 1\), or \(2x^3 - 3\), or \(2x^3 +\) any constant, also gives \(6x^2\). We cannot determine the constant without extra information, so we must write:

\[y = 2x^3 + c\]

The value \(c\) is the constant of integration. The result is called an indefinite integral because \(c\) is unknown.

Notation

The integral of \(6x^2\) with respect to \(x\) is written:

\[\int 6x^2\,dx = 2x^3 + c\]

2. Indefinite Integration

General Power Rule

\[\int ax^n\,dx = \frac{ax^{n+1}}{n+1} + c \qquad (n \neq -1)\]

A constant factor can be moved outside the integral:

\[\int ax^n\,dx = a\int x^n\,dx\]

Worked Examples

EXAMPLE A

Find \(\displaystyle\int 12x^5\,dx\)

\[\int 12x^5\,dx = \frac{12x^6}{6} + c = 2x^6 + c\]
EXAMPLE B

Find \(\displaystyle\int 8\,dx\)

\[\int 8\,dx = 8x + c\]
EXAMPLE C

Find \(\displaystyle\int \frac{x^3}{2}\,dx\)

\[\int \frac{x^3}{2}\,dx = \frac{1}{2}\cdot\frac{x^4}{4} + c = \frac{x^4}{8} + c\]

Integration of Polynomials

Integrate term by term:

\[\int [f(x) + g(x)]\,dx = \int f(x)\,dx + \int g(x)\,dx\]
EXAMPLE — POLYNOMIAL

Find \(\displaystyle\int (8x^3 + 5x^2 - 6x + 2)\,dx\)

\[ = \frac{8x^4}{4} + \frac{5x^3}{3} - \frac{6x^2}{2} + 2x + c = 2x^4 + \frac{5x^3}{3} - 3x^2 + 2x + c \]

Negative and Fractional Powers

The power rule applies to all rational \(n\) except \(n = -1\).

EXAMPLE — NEGATIVE POWER

Find \(\displaystyle\int \frac{2}{x^2}\,dx\)

Write as \(2x^{-2}\), then:

\[\int 2x^{-2}\,dx = \frac{2x^{-1}}{-1} + c = -\frac{2}{x} + c\]
EXAMPLE — FRACTIONAL POWER

Find \(\displaystyle\int \!\left(3\sqrt{x} + \frac{2}{\sqrt{x}}\right)dx\)

Write as \(3x^{1/2} + 2x^{-1/2}\), then:

\[ = \frac{3x^{3/2}}{3/2} + \frac{2x^{1/2}}{1/2} + c = 2x^{3/2} + 4x^{1/2} + c \]

3. Rewriting Expressions Before Integrating

Some expressions must be rewritten as separate terms of the form \(ax^n\) before the power rule can be applied. The two most common techniques are expanding brackets and splitting a fraction.

Expanding Brackets First

EXAMPLE

Find \(\displaystyle\int (3 - 2x)^2\,dx\)

Step 1 — expand:

\[(3 - 2x)^2 = 9 - 12x + 4x^2\]

Step 2 — integrate term by term:

\[ \int (9 - 12x + 4x^2)\,dx = 9x - \frac{12x^2}{2} + \frac{4x^3}{3} + c = 9x - 6x^2 + \frac{4x^3}{3} + c \]

Splitting a Fraction

EXAMPLE

Find \(\displaystyle\int \frac{4x^3 - 5x^2 + 2}{x^2}\,dx\)

Step 1 — divide each term by \(x^2\):

\[\frac{4x^3}{x^2} - \frac{5x^2}{x^2} + \frac{2}{x^2} = 4x - 5 + 2x^{-2}\]

Step 2 — integrate:

\[ \int (4x - 5 + 2x^{-2})\,dx = 2x^2 - 5x + \frac{2x^{-1}}{-1} + c = 2x^2 - 5x - \frac{2}{x} + c \]

4. The Natural Logarithm Rule

When \(n = -1\) the power rule fails. Instead we have the special case:

\[\int \frac{1}{x}\,dx = \ln|x| + c\]

More generally, when the numerator is the derivative of the denominator:

\[\int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + c\]

How to Recognise It

Ask: "Is the numerator the derivative (or a constant multiple of the derivative) of the denominator?"

If yes, the result is \(\ln|\text{denominator}| + c\) (adjusting for the constant).

Worked Examples

EXAMPLE A — SIMPLE LINEAR

Find \(\displaystyle\int \frac{1}{x-1}\,dx\)

The denominator is \(x-1\) and \(\dfrac{d}{dx}(x-1) = 1\), which equals the numerator.

\[\int \frac{1}{x-1}\,dx = \ln|x-1| + c\]
EXAMPLE B — TRIGONOMETRIC (TAN X)

Find \(\displaystyle\int \tan x\,dx = \int \frac{\sin x}{\cos x}\,dx\)

The denominator is \(\cos x\) and \(\dfrac{d}{dx}(\cos x) = -\sin x\). The numerator \(\sin x = -1 \times (-\sin x)\):

\[\int \frac{\sin x}{\cos x}\,dx = -\int \frac{-\sin x}{\cos x}\,dx = -\ln|\cos x| + c\]

This can also be written as \(\ln|\sec x| + c\).

EXAMPLE C — POLYNOMIAL

Find \(\displaystyle\int \frac{2x}{x^2+5}\,dx\)

The denominator is \(x^2+5\) and \(\dfrac{d}{dx}(x^2+5) = 2x\), which is exactly the numerator:

\[\int \frac{2x}{x^2+5}\,dx = \ln|x^2+5| + c = \ln(x^2+5) + c\]

(Absolute value bars are optional here since \(x^2+5 > 0\) always.)

EXAMPLE D — ADJUSTING CONSTANTS

Find \(\displaystyle\int \frac{3}{2x+7}\,dx\)

\(\dfrac{d}{dx}(2x+7) = 2\), but the numerator is \(3\). Factor:

\[\int \frac{3}{2x+7}\,dx = \frac{3}{2}\int \frac{2}{2x+7}\,dx = \frac{3}{2}\ln|2x+7| + c\]

5. Finding the Constant of Integration Given a Point

If we know both the gradient function \(\dfrac{dy}{dx}\) and a point \((x_0,\, y_0)\) that lies on the curve, we can determine \(c\) exactly by substituting the coordinates after integrating.

Method

  1. Integrate \(\dfrac{dy}{dx}\) to get \(y = F(x) + c\).
  2. Substitute the given point \((x_0, y_0)\) into the equation.
  3. Solve for \(c\).
  4. Write the final equation of the curve with the exact value of \(c\).

Worked Example

EXAMPLE

A curve \(y = f(x)\) passes through the point \((2, 9)\). Given that \(\dfrac{dy}{dx} = 8x^3 - 10x\), find the equation of the curve.

Step 1 — integrate:

\[ y = \int (8x^3 - 10x)\,dx = \frac{8x^4}{4} - \frac{10x^2}{2} + c = 2x^4 - 5x^2 + c \]

Step 2 — substitute \((2, 9)\):

\[ 9 = 2(2)^4 - 5(2)^2 + c = 32 - 20 + c = 12 + c \] \[c = -3\]

Answer:

\[y = 2x^4 - 5x^2 - 3\]

6. Examination-Style Questions

QUESTION 1

Find \(\displaystyle\int (5x^4 - 3x^2 + 7)\,dx\).

\[\int (5x^4 - 3x^2 + 7)\,dx = x^5 - x^3 + 7x + c\]
QUESTION 2

Find \(\displaystyle\int \frac{3}{x^3}\,dx\).

Write \(3x^{-3}\):

\[\int 3x^{-3}\,dx = \frac{3x^{-2}}{-2} + c = -\frac{3}{2x^2} + c\]
QUESTION 3

Find \(\displaystyle\int (2 + 3x)^2\,dx\).

Expand: \((2+3x)^2 = 4 + 12x + 9x^2\)

\[\int (4 + 12x + 9x^2)\,dx = 4x + 6x^2 + 3x^3 + c\]
QUESTION 4 — EXAM STYLE

A curve has gradient function \(f'(x) = 6x^2 - 7\). Given that the curve passes through the point \((2,\, 5)\), find the equation of the curve.

Step 1 — integrate:

\[f(x) = 2x^3 - 7x + c\]

Step 2 — substitute \((2, 5)\):

\[5 = 2(8) - 7(2) + c = 16 - 14 + c = 2 + c \implies c = 3\]

Answer:

\[f(x) = 2x^3 - 7x + 3\]
QUESTION 5

Find \(\displaystyle\int \frac{6x^4 - 9x^3 + 3x}{3x^2}\,dx\).

Split the fraction:

\[\frac{6x^4}{3x^2} - \frac{9x^3}{3x^2} + \frac{3x}{3x^2} = 2x^2 - 3x + x^{-1}\] \[\int \!\left(2x^2 - 3x + \frac{1}{x}\right)dx = \frac{2x^3}{3} - \frac{3x^2}{2} + \ln|x| + c\]

7. Quick Reference — Lecture 1 Rules

Rule Formula Condition
Power Rule \(\displaystyle\int ax^n\,dx = \dfrac{ax^{n+1}}{n+1} + c\) \(n \neq -1\)
Constant \(\displaystyle\int k\,dx = kx + c\) —
Constant Factor \(\displaystyle\int k\,f(x)\,dx = k\int f(x)\,dx\) —
Sum / Difference \(\displaystyle\int [f \pm g]\,dx = \int f\,dx \pm \int g\,dx\) —
Expand First Expand brackets, then integrate each term —
Split Fraction Divide every numerator term by denominator, then integrate —
Find \(c\) Integrate, then substitute the given point to solve for \(c\) Point must be given
💡 Lecture 1 Tips:
  • Always include \(+c\) — omitting it loses marks in every indefinite integral.
  • Rewrite negative and fractional powers using index notation before applying the rule.
  • You cannot integrate a product or quotient directly — expand or split first.
  • The power rule fails at \(n = -1\); \(\int x^{-1}\,dx = \ln|x| + c\).
  • When finding the constant: integrate first, substitute the point second, solve for \(c\) third.
