CALCULUS 2 SOLUTIONS

INTEGRATION PRACTICE - LECTURE 1

Integration Practice

Problem 1
\[ \int 2x^m \, dx \]
Apply the general power rule for integration: \( \int ax^n \, dx = \frac{ax^{n+1}}{n+1} \)
\[ = \frac{2x^{m+1}}{m+1} + C \]
Problem 2
\[ \int 2x^4 \, dx \]
Apply the power rule. Increase the exponent by 1 (4 + 1 = 5) and divide by the new exponent:
\[ = 2 \left( \frac{x^5}{5} \right) \]
\[ = \frac{2}{5}x^5 + C \]
Problem 3
\[ \int 2x^{-3} \, dx \]
Apply the power rule. Increase the exponent by 1 (-3 + 1 = -2) and divide by the new exponent:
\[ = 2 \left( \frac{x^{-2}}{-2} \right) \]
Simplify the expression:
\[ = -\frac{1}{x^2} + C \]
Problem 4
\[ \int \frac{5}{\sqrt{x}} \, dx \]
Rewrite the radical as a fractional exponent:
\[ \int 5x^{-1/2} \, dx \]
Apply the power rule. Increase the exponent by 1 (-1/2 + 1 = 1/2) and divide by the new exponent:
\[ = 5 \left( \frac{x^{1/2}}{1/2} \right) = 5 \times 2x^{1/2} \]
Rewrite back into radical form:
\[ = 10\sqrt{x} + C \]
Problem 5
\[ \int (\frac{2}{\sqrt{y}} + \frac{3}{y^2}) \, dy \]
Rewrite terms using negative and fractional exponents:
\[ \int (2y^{-1/2} + 3y^{-2}) \, dy \]
Integrate term by term applying the power rule:
\[ = 2 \left( \frac{y^{1/2}}{1/2} \right) + 3 \left( \frac{y^{-1}}{-1} \right) \]
Simplify the terms:
\[ = 4y^{1/2} - 3y^{-1} \]
\[ = 4\sqrt{y} - \frac{3}{y} + C \]
Problem 6
\[ \int (x-1)(x-2) \, dx \]
Expand the binomials before integrating:
\[ (x-1)(x-2) = x^2 - 3x + 2 \]
Integrate the expanded polynomial term by term:
\[ \int (x^2 - 3x + 2) \, dx \]
Apply the power rule to each term:
\[ = \frac{1}{3}x^3 - \frac{3}{2}x^2 + 2x + C \]
Problem 7
\[ \int \frac{3x^2 - x + 1}{2x^4} \, dx \]
Split the fraction to divide each term in the numerator by the denominator:
\[ \int \left( \frac{3x^2}{2x^4} - \frac{x}{2x^4} + \frac{1}{2x^4} \right) \, dx \]
Simplify each term using exponent rules:
\[ \int \left( \frac{3}{2}x^{-2} - \frac{1}{2}x^{-3} + \frac{1}{2}x^{-4} \right) \, dx \]
Integrate term by term:
\[ = \frac{3}{2}\left(\frac{x^{-1}}{-1}\right) - \frac{1}{2}\left(\frac{x^{-2}}{-2}\right) + \frac{1}{2}\left(\frac{x^{-3}}{-3}\right) \]
Simplify the coefficients and rewrite with positive exponents:
\[ = -\frac{3}{2x} + \frac{1}{4x^2} - \frac{1}{6x^3} + C \]
Problem 8
\[ \int \frac{x^2 - x - 6}{x+2} \, dx \]
Factor the quadratic in the numerator:
\[ x^2 - x - 6 = (x-3)(x+2) \]
Cancel the common factor \( (x+2) \):
\[ \int \frac{(x-3)(x+2)}{x+2} \, dx = \int (x-3) \, dx \]
Integrate term by term:
\[ = \frac{1}{2}x^2 - 3x + C \]
Problem 9
\[ \int \frac{x^3 - 27}{x-3} \, dx \]
Factor the difference of cubes in the numerator:
\[ x^3 - 27 = x^3 - 3^3 = (x-3)(x^2 + 3x + 9) \]
Cancel the common factor \( (x-3) \):
\[ \int \frac{(x-3)(x^2 + 3x + 9)}{x-3} \, dx = \int (x^2 + 3x + 9) \, dx \]
Integrate term by term using the power rule:
\[ = \frac{1}{3}x^3 + \frac{3}{2}x^2 + 9x + C \]
Problem 10
\[ \int \cos 10x \, dx \]
Use the standard integration rule for cosine with a linear argument: \( \int \cos(ax) \, dx = \frac{1}{a}\sin(ax) \)
Here, \( a = 10 \):
\[ = \frac{1}{10}\sin 10x + C \]
Problem 11
\[ \int \sec^2 5x \, dx \]
Use the standard integration rule for secant squared with a linear argument: \( \int \sec^2(ax) \, dx = \frac{1}{a}\tan(ax) \)
Here, \( a = 5 \):
\[ = \frac{1}{5}\tan 5x + C \]
Problem 12
\[ \int (\frac{\sec^2 x}{3} - 2\sec 4x \tan 4x) \, dx \]
Split into two separate integrals:
\[ \int \frac{1}{3}\sec^2 x \, dx - \int 2\sec(4x)\tan(4x) \, dx \]
Integrate the first term using \( \int \sec^2 x \, dx = \tan x \):
\[ \frac{1}{3}\tan x \]
Integrate the second term using \( \int \sec(ax)\tan(ax) \, dx = \frac{1}{a}\sec(ax) \) with \( a = 4 \):
\[ -2 \left( \frac{1}{4}\sec(4x) \right) = -\frac{1}{2}\sec(4x) \]
Combine the results:
\[ = \frac{1}{3}\tan x - \frac{1}{2}\sec 4x + C \]
Problem 13
\[ \int \frac{\sin^2 x}{1 - \cos x} \, dx \]
Use the Pythagorean trigonometric identity \( \sin^2 x = 1 - \cos^2 x \):
\[ \int \frac{1 - \cos^2 x}{1 - \cos x} \, dx \]
Factor the numerator as a difference of squares:
\[ \int \frac{(1 - \cos x)(1 + \cos x)}{1 - \cos x} \, dx \]
Cancel the common factor \( (1 - \cos x) \):
\[ \int (1 + \cos x) \, dx \]
Integrate term by term:
\[ = x + \sin x + C \]
Problem 14
\[ \int (2x + 5)^3 \, dx \]
Use the linear composite power rule: \( \int (ax+b)^n \, dx = \frac{1}{a} \cdot \frac{(ax+b)^{n+1}}{n+1} \)
Here, \( a = 2 \), \( b = 5 \), and \( n = 3 \):
\[ = \frac{1}{2} \cdot \frac{(2x+5)^4}{4} \]
Multiply the constants together:
\[ = \frac{1}{8}(2x+5)^4 + C \]
